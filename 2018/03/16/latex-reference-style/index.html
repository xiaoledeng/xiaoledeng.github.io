LaTeX修改参考文献样式

2018-03-16
#LaTeX

前言

使用LaTeX写学位论文中，会有各种不同的参考文献样式。

关于LaTeX使用参考文献的基础知识，可以参考已写的两篇Blog：

  • 终极参考文献引用管理—BibTeX
  • LaTeX参考文献管理工具BibDesk

本文简要介绍如何在更换参考文献的样式。

命令

  • 在tex主文件同目录下面，要有natbib.bst；当然也可以换成其他的参考文献格式，要提前下载这个bst文件。

  • 在tex主文件开头，添加参考文献格式natbib的声明：

\usepackage{natbib}
  • 在tex主文件放参考文献的位置，添加引用参考文献的说明：
\bibliographystyle{natbib}
\bibliography{mybibtex}

注意：mybibtex.bib为参考文献BibTeX的一个文件，应放在tex主文件的同目录下。

  • 在tex主文件内容中，引用文献的格式为：
\citet{...}
\citep{...}
