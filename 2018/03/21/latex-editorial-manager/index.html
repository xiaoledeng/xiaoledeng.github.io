LaTeX文稿投稿到Editorial-Manager系统

2018-03-21
#LaTeX

继 LaTeX 投稿 ScholarOne Manuscript 之后，本次是LaTeX文稿投稿到Editorial Manager系统，从而由LaTeX文件在投稿系统中生成PDF文件。

1. 流程

首先，点击下图中的Submit New Manuscript，进入到New Submission里面的细节。

而后按照New Submission的流程，把一步一步都填写完整并正确，分别有：

  • Select Article Type;
  • Enter Title;
  • Add/Edit/Remove Authors;
  • Funding Information;
  • Select Section/Category;
  • Submit Abstract;
  • Enter Keywords;
  • Select Classifications;
  • Enter Comments;
  • Suggest Reviewers;
  • Attach Files;

2. 上传LaTeX文稿的注意事项

在Attach Files这一步，上传LaTeX文稿以及图片。

  • 首次，上传LaTeX文稿，Item为*Manuscript，Description为Manuscript，此为序号1；
  • 其次，上传图片文件，Item为Figure，Description为Fig1.eps，Fig2.eps等；

完成上面两个步骤之后，可以往下面的步骤进行，一直到building PDF这一步，生成正确的PDF文件。

注意事项：

  • 图片文件Fig1.eps等，应该与LaTeX文稿在同一个路径下；
  • 在LaTeX文稿中，引用的图片文件名应该等于上传图片的文件名；
  • 本地的格式文件（例如，参考文献引用格式—bst文件，文章属性格式—cls和clo文件），则不需要上传到投稿系统里面；

3. 检查从LaTeX文稿到PDF文件是否正确

在Submissions Waiting for Author’s Approval一栏中，查看最后的PDF是否生成成功，检查列表参考：SCI论文投稿Submissions Waiting for Author’s Approval的检查清单。
