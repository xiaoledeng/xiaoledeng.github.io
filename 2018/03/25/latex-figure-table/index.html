LaTeX目录中加入插图和表格命令

2018-03-25
#LaTeX

前言

LaTeX学位论文的目录中，会有插图、表格和术语表的大标题。

本文简要介绍如何在目录中加入学位论文中出现的插图、插表和术语表的目录。

命令

  • 插图列表：
\renewcommand*{\listfigurename}{插图列表}
\listoffigures
\addcontentsline{toc}{chapter}{插图列表}
  • 表格列表：
\renewcommand*{\listtablename}{表格列表}
\listoftables
\addcontentsline{toc}{chapter}{表格列表}
  • 术语缩写列表：
\include{includefile/Abbreviation}
\addcontentsline{toc}{chapter}{术语缩写列表}
