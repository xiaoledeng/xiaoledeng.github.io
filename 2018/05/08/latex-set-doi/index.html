LaTeX设置doi

2018-05-08
#LaTeX

在用LaTeX写论文的时候，参考文献里面的doi号的，点击doi号之后，外在的链接可以打开这一篇文献。

方法一

使用新命令：

\newcommand{\DOI}[1]{doi: \href{https://doi.org/#1}{#1}}

在需要显示doi的地方，使用以下命令

\DOI{10.1007/s10712-018-9460-8}

方法二

使用bib文献库的方法：

前面添加包：

\usepackage{natbib}
\usepackage{doi}

在引用参考文献的地方：

\bibliographystyle{spbasic}

注意：bib文献库里面，doi号码要有。
