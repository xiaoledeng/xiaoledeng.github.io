Visual Studio Code编辑器

2021-04-12 10:00
#Linux #Lists #Skills

1. 简介

Visual Studio Code是一款编辑器，官网为：Visual Studio Code - Code Editing. Redefined。

特色是，可以把常用的编程环境都能设置，即All in one。

2. 常用插件清单

  • Auto Rename Tag：自动写好tag
  • Better Comments：设置注释的各种格式
  • Bracket Pair Colorizer：设置不同颜色的括号
  • Browser Preview：打开网页链接
  • Code Runner：运行各种语言，包括：C、C++、Python、Julia等
  • Code Spell Checker：语法检查
  • Draw.io Integration：流程图
  • fortran：Fortran支持
  • LaTeX Workshop：LaTeX支持
  • Markdown All in One：Markdown支持
  • …

3. Mac设置命令行缩写code

首先，打开~/.zshrc或~/.bashrc；

而后，添加以下命令：

alias code=\''/Applications/Visual Studio Code.app/Contents/MacOS/Electron'\'

在终端输入source ~/.zshrc或source ~/.bashrc，而后输入code即可打开vscode编辑器。
