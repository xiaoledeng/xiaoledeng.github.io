Ubuntu配置LaTeX环境

2021-12-08
#Unix #Experiences #LaTeX #Ubuntu #Software

1. 安装Tex Live和Texmaker

sudo apt-get install texlive-full
sudo apt-get install texmaker

注意：软件源可以改为国内中科大的源，这样下载速度会很快，Tex Live大约有2.98G。

部分过程如下：

0 upgraded, 346 newly installed, 0 to remove and 16 not upgraded.
Need to get 3,059 MB of archives.
After this operation, 5,873 MB of additional disk space will be used.
...
Fetched 3,059 MB in 1min 42s (30.0 MB/s)

2. 安装vscode

打开Ubuntu Software，搜索code，点击安装。

打开vscode，安装插件LaTex Workshop，之后就可以在vscode里面编译latex文件了。
