1. 参考文献管理的思考

从最开始进入科研领域，参考文献（一个个PDF文件）成了研究的基础。

从2013年至今，在NoteExpress、Endnote、Mendeley、Zoter和Paper等众多参考文献管理软件中，每一个都没有坚持用下来。反而是最简单的管理方式——文件夹+PDF文献一直在坚持使用，而且效果还不错，是挺适合自己的一种方式。

博士阶段的主要创新点是在硕士毕业2016年的那个暑假，将所有的参考文献整理一遍之后，突然有亮光。我称之为All in one文献管理法。

All in one文献管理法的步骤

  1. 拷贝：复制所有的电子文献的PDF文件，并黏贴到一个文件夹目录下；
  2. 重命名：对每个PDF文件进行规范化重命名，格式为：作者.年.期刊缩写或book或phd或master；
  3. 检索：利用Linux的ls或ll命令操作；
  4. 阅读：定位到要看的文献，用PDF阅读器看；

粗略估算，参考文献的PDF文件有3000左右。

后续还要花费大量时间对PDF文件进行规范化的重命名操作，这是一个大工程，同时也是一个机遇——新的创新点酝酿和出现。
