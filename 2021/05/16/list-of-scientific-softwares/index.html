macOS科研软件清单

2021-05-16 21:00
#Lists #Linux #Experiences

1. 前言

本人的专业是理工科，科研方面会用到一些常用的软件，有必要撰写macOS系统的科研软件清单，以供自己总结，也可给看到的人参考。

2. 软件清单

科研写作

  • Texpad：macOS系统下的LaTeX编译器
  • MacTeX：整套macOS下的LaTeX的环境配置
  • 公式利器Mathpix Snipping Tool：可把截图公式转变为LaTeX代码
  • Visual Studio Code：稳定的文本编辑器，可替换Sublime 3、Typora、Github Desktop、GMT、draw.io、Python和Fortran
  • Word、PPT和Excel：Office和WPS
  • Grammarly：英文语法检查工具
  • 搜狗输入法：输入法

画图

  • GMT：用于绘制论文图片，编程代码可重复
  • Ximd Zen：思维导图
  • draw.io：流程图

科学计算和编程

  • Mathematica：用于公式推导
  • Matlab：用于数值运算
  • Anaconda：Python软件包
  • ifort：Fortran编译器

上网和服务器

  • ClashX：macOS系统下的上网工具1
  • V2RayX：macOS系统下的上网工具2
  • Tunnelblick：教育网网络访问服务器
  • Royal TSX：服务器上传和下载文件

远程访问

  • TeamViewer
  • Anydesk
  • Todesk
  • ZeroTier one

线上会议

  • Zoom
  • TencentMeeting
  • Skype

备份

  • OneDrive：教育用户有1T容量的网盘
  • Syncthing：资料备份工具
  • Paragon NTFS for Mac：硬盘管理工具

其他软件

  • Keka：解压和压缩文件
  • iTerm：命令行工具
  • Homebrew：命令行工具
  • Google Chrome：浏览器
  • PDF Expert：PDF阅读器
  • Parallels Desktop：虚拟机，安装Windows系统
