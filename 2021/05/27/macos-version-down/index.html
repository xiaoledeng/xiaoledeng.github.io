3. 如何降级系统？

此部分内容来自：macOS Big Sur降级的三种方法。

  1. 下载系统：在How to get old versions of macOS 下载 OS X El Capitan 10.11 ；

  2. 制作系统盘：依次打开「应用程序」-「实用工具」-「终端」，然后输入一下代码并按下回车：

sudo /Applications/Install\ OS\ X\ El\ Capitan.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume --applicationpath /Applications/Install\ OS\ X\ El\ Capitan.app

这里 OS\ X\ El\ Capitan.app 是下载的系统版本，如果下载的并非 OS\ X\ El\ Capitan，需要改成对应的系统名。「MyVolume」是 U 盘的名称，如果 U 盘不是这个名字，将命令中的「MyVolume」改成 U 盘名称即可。

也可以在Windows下用软碟通，通过写入硬盘镜像的方式制作启动 U 盘。

  1. 抹掉：先重启系统，按住 Command 键，启动选择界面，通常来说第一个是系统盘，后面的才是 U 盘系统，这里选择第二个并等到 macOS 实用工具的界面出来，点击磁盘工具。在磁盘工具的页面，找到系统盘，选中并点击上方的「抹掉」，格式选择 APFS，点击抹掉。（抹掉前记得备份）这里不建议选择区分大小写的格式，会导致一部分的软件无法使用，比如 Steam。

  2. 重新安装 macOS：等到抹掉这个过程完成后，返回最初 macOS 「实用工具」的界面，选中「重新安装 macOS」，接着按照系统提示进行即可，整个过程大约需要半小时左右。
