学习方法—书单

2020-03-03
#Booklist #Education

1. 终身学习

  • 《科学学习：斯坦福黄金学习法则-丹尼尔 L.施瓦茨等》：提供了经过科学验证的26种学习法则，可以用于设计个性化学习方案，激发学生的兴趣和动机，提升学生的自主学习能力，为面向未来的成长发展打下坚实的基础。

  • 《刻意学习-Scalers》

  • 《有效学习-乌尔里希·伯泽尔》

  • 《学会学习-斋藤孝》

  • 《深度学习：彻底解决你的知识焦虑-今井睦美》

  • 《终身学习：10个你必须掌握的未来生存法则-丹·苏利文》

  • 《受益一生的41种学习方法-吴光远》

  • 《思维导图完整手册-东尼·博赞》

  • 《学习之道-芭芭拉·奥克利》

  • 《学会如何学习-芭芭拉·奥克利 特伦斯·谢诺夫斯基 阿利斯泰尔·麦康维尔》

  • 《7S倍速学习法-高效能学习研究机构》

  • 《超级学霸：受用终身的速效学习法-奥拉夫·舍韦》

  • 《学霸养成记：开启学习力的10把金钥匙-林宗霖》：10把金钥匙一意义、 动力、努力、坚持、心态、自我管理、养成习惯、相信自己、亲密关系、精力管理，即可开启学习力之门，达到提升成绩的目标。

2. 职场学习

  • 《学习力：颠覆职场学习的高效方法-王世民 缪志聪》

3. 大学学习

  • 《如何高效学习：1年完成麻省理工4年33门课程的整体性学习法-斯科特·扬》
  • 《考研记：备考成功之道-小莫》
  • 《东大教授教我的学习法-永野裕之》： 学习诀窍包括：接近本质的最快思考法（PART 1）、目标的算法（PART 2）、问题集的使用方法（PART 3）。另外，“记忆法（PART 4）”也有所提及。
  • 《如何学习-本尼迪克特·凯里》
  • 《卓有成效的学习方法：康奈尔大学最受欢迎的学习课-爱德华·李·桑代克》

4. 高中学习

  • 《走向北大清华：优等生高效学习方法全集-曾艳蓓编著》：主要从时间管理、高效听课、巧妙做题、分科学习、加强记忆、课外阅读、应试技巧、心态调整等八个方面(共80个方法)》着手，针对中学生的学习特点和学习习惯，收录了近百名从北京四中、黄冈中学、人大附中等重点中学升入北京大学、清华大学的学子的亲身经历和感受，从不同的角度讲述了他们的成长经历、学习方法、备考经验和应试技巧。这些都是经过实践检验的宝贵经验，能有效解决大家在学习中遇到的问题，明显提高大家的学习成绩。

  • 《哈佛优等生的学习方法和培养细节-廉勇》：以哈佛优等生的学习方法和家庭教育为素材，从兴趣、时间管理、阅读、综合发展、自由成长、家庭环境、赏识、自信、独立性等方面出发，全方位、多角度地对哈佛优等生的学习方法和培养细节进行了细致的介绍和分析，旨在为广大青少年和家长朋友们提供一个可以借鉴和学习的机会。

  • 《中学生高效学习方法指导-罗天翔编著》

  • 《学习力：我是这样考上清华的-陈陆淼》：全书共分为12章，前面6章讲解了宏观的学习方法，包括认清自己、对高考的错误认识、有效利用时间、学渣如何逆袭、错题本的运用、快速记忆方法等内容。后面6章针对具体科目，讲解了答题技巧，这些技巧你可能在其他辅导书上也看过，但都是作为旁门左道来讲，而本书旨在提高考试成绩，同时训练思维能力，相信能给你不一样的感觉。

  • 《清华北大状元告诉你的73个最优秀学习习惯-盛建武 张建华》：从以下几个方面帮助大家养成良好的学习习惯：有疑必问的学习习惯，专注的学习习惯，有效记忆的学习习惯，科学利用时间的学习习惯，预习复习的学习习惯，高效的课堂学习习惯，各个学科的学习习惯以及阅读和考试习惯。

5. 初中学习

  • 《数学好的人是如何思考的-永野裕之》：初中数学

6. 小学学习

  • 《小学生学习成绩暴增法：孩子最不会抵触的家庭作业辅导法-坂本七郎》：家庭作业辅导法，也可以叫“犹太式辅导法”，每周只需要3到4次，每次20分钟，就能取得明显效果。
  • 《小学六年，这样学习最高效：优秀学生都在用的67个学习方法-石娟》
