校园无线网连接DNS问题

2020-09-16
#Experiences

1.简介

南方科技大学的无线网络，有5G WiFi，速度还是不错的。

登录的前提是，需要有学校的CAS(Central Authentication Service)账号。

2.问题描述

连接南方科技大学无线网络SUSTech-wifi-5G后，弹出选择连接方式，点击CAS登录后，没有反应，弹不出接下来的登录账号的界面。

3.问题解决

这里介绍macOS系统下面的设置方法，Windows类似。

  • 点击System Preferences—Network—WiFi Advanced—DNS
  • 添加新的DNS号：172.18.1.92和172.18.1.93，注意：原有的192.168.100.1和192.168.1.1继续保持
  • 点击OK—Apply
