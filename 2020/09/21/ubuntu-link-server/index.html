Ubuntu连接服务器的账号登录和上传下载文件

2020-09-21 12:00
#Skills #Linux

1.账号登录

Ubuntu打开终端，输入：

ssh user_name@ip_address

注释：

  • user_name为用户名
  • ip_address为服务器的ip地址
  • 如果有端口设置，加入 -p xx，xx为端口号。

2.上传下载文件

本地上传到服务器：

scp /path_local_file user_name@ip_address:/path_server_folder

例如：

scp /home/shalom/a.txt deng@127.168.1.1:/home/deng/data/

解释：

  • path_local_file为本地文件的地址
  • /path_server_folder为服务器上路径
  • 如果有端口设置，加入 -p xx，xx为端口号。
  • 如果是本地文件夹，加 -r

服务器下载到本地：

scp user_name@ip_address:/path_server_file /path_local_folder
