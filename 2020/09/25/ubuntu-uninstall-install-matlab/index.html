Ubuntu卸载和安装MATLAB

2020-09-25
#Experiences #Unix #Ubuntu

1.Ubuntu卸载matlab

查看MATLAB安装路径：

$ whereis matlab
matlab: /usr/local/bin/matlab

卸载MATLAB：

sudo rm -rf /usr/local/MATLAB/R2020b/
sudo rm /usr/local/bin/matlab

清理配置残留：

sudo apt-get autoremove
sudo apt-get clean

2.安装

sudo ./install

第一次启动需要执行命令:

sudo matlab

添加MATLAB的支持：

sudo apt install matlab-support
