Ubuntu安装Sublime和latex-tools

2020-09-17
#Unix #Experiences #Ubuntu

1.安装Sublime text

  • Sublime Text中文文档之 Linux Package Manager仓库 - Sublime Text 中文官网

安装GPG密钥：

wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -

确保apt已设置为使用https源：

sudo apt-get install apt-transport-https

选择要使用的频道：

  • Stable
echo "deb https://download.sublimetext.com/ apt/stable/" | sudo tee /etc/apt/sources.list.d/sublime-text.list
  • Dev
echo "deb https://download.sublimetext.com/ apt/dev/" | sudo tee /etc/apt/sources.list.d/sublime-text.list

更新apt源并安装Sublime Text

sudo apt-get update
sudo apt-get install sublime-text

2.安装Sublime Text3插件latex-tools

  • 在终端输入sublime 启动sublime text3
  • 按快捷键Ctrl+Shift+P启动在对话框输入Install Package回车，然后在插件安装对话框输入latex tools就能看到，选中安装。
  • 打开Perference->Package Settings->LaTexTools->Settings-User
  • 在打开文件的227行"linux"中配置你的TexLive路径 “texpath” : “$PATH:/usr/local/texlive/2020/bin/x86_64-linux”
  • 设置487中 “viewer”: “evince”（不设置空着应该也可以，系统默认用自己的evince）。
  • 安装dbus及依赖
sudo apt-get install dbus libdbus-glib-1-dev gcc
