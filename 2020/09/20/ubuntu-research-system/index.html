Ubuntu 20.04 LTS作为科研主系统

2020-09-20
#Experiences #Unix #Ubuntu

1. 简介

新台式机电脑是dell，默认安装的是Windows10，可是运行速度堪忧，完全跟不上鼠标和键盘操作的节奏。

是故，决定安装Ubuntu 20.4 LTS系统作为科研的主系统。

台式机的基本配置：

  • 内存：16G+16G=32G
  • 处理器：Intel® Core™ i7-8700 CPU @ 3.20GHz × 12
  • 图形：llvmpipe (LLVM 10.0.0, 256 bits) / llvmpipe (LLVM 10.0.0, 256 bits)
  • 磁盘空间：1.0TB

Ubuntu系统的基本配置：

  • OS name：Ubuntu 20.04.1 LTS
  • OS Type：64位
  • GNOME Version：3.36.3
  • Windowing System：X11

2. 录制U盘启动盘

在USTC Open Source Software Mirror下载Ubuntu 20.4 LTS的镜像文件，而后刻录到U盘里面，作为系统系统盘。

插上U盘，重启电脑，按F12，从U盘启动，依次按照步骤安装Ubuntu20.04LTS。

3. 基本软件

首先，更改软件源：Ubuntu更换中科大教育网的软件源，以加快软件的下载速度。

其次，依次安装常用软件：

  • Mathematica12.1.1，通过Mathematica 激活指南下载和安装
  • 谷歌浏览器，在Google Chrome 网络浏览器-其他平台-linux下载最新的Google Chrome .deb软件包，而后安装即可
  • sublime编辑器，见Ubuntu安装Sublime和latex-tools
  • texlive2020，见Ubuntu安装Texlive2020
  • texstudio，见TeXstudio- Download - Linux - Ubuntu 20.04
  • gmt 6.1.1，见Linux/macOS 下编译GMT源码 — GMT中文手册
  • mathpix，见Install Mathpix Snipping Tool on Ubuntu using the Snap Store | Snapcraft
  • Teamviewer，见下载 Linux 版 TeamViewer，进行远程桌面访问和协作
  • Anydesk，见Linux上的AnyDesk远程桌面应用程序
  • 搜狗输入法，见搜狗输入法 for linux
  • Master PDF，见Master PDF Editor for Linux. Download Master PDF Editor for Linux.
  • Word系列，卸载libreoffice：sudo apt-get remove libreoffice-common；安装WPS，见：WPS Office 2019 For Linux官网产品,WPS For Linux版下载，办公从未如此轻松
  • unrar解压，命令：sudo apt-get install unrar
  • 删掉基本不用的自带软件（用的时候再装也来得及）：
sudo apt-get remove thunderbird totem rhythmbox empathy brasero simple-scan gnome-mahjongg aisleriot gnome-mines cheese transmission-common gnome-orca  gnome-sudoku

sudo apt-get remove onboard deja-dup
  • JabRef参考文献，见Ubuntu安装JabRef
  • Snap管理软件，见Linux管理软件Snap
