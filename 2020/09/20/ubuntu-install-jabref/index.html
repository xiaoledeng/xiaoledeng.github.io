Ubuntu安装JabRef

2020-09-20
#Unix #Experiences #Ubuntu

1.简介

Ubuntu下，对于Bib文件格式，需要用JabRef进行打开，即进行文献管理操作。

2.下载安装

第一种方法：

sudo snap install jabref

第二种方法：在Download JabRef latest release页面，下载JabRef Debian and Ubuntu版本，并安装。
