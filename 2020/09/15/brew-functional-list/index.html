地址：The Missing Package Manager for macOS (or Linux) — Homebrew

2.基本操作

  • 搜索：brew search xxx
  • 安装：brew install xxx
  • 查询：brew info xxx
  • 更新：brew update
  • 检查是否有新版本：brew outdated
  • 升级：brew upgrade xxx
  • 清理：brew cleanup xxx
  • 删除：brew uninstall xxx
  • 安装列表：brew list
