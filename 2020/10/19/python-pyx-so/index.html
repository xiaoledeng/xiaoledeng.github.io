Python的.pyx文件编译生成.so文件

2020-10-19 15:00
#Python

文件setup.py的内容为：

from distutils.core import setup
from Cython.Build import cythonize
import numpy

setup(
    ext_modules = cythonize("add_num.pyx"),
    include_dirs=[numpy.get_include()]
)

add_num.pyx为输入文件。

运行：

python setup.py build_ext --inplace

即可生成add_num.so文件。
