Anaconda3安装python2.7环境

2020-10-14
#Python #Software

1.简介

目前，主流的Python环境是python3，利用Anaconda3可以安装python3相关的工具包。

Anaconda的官网下载链接为：Anaconda installer archive

如果需要python2.7的环境，可以利用Anaconda3来虚拟python2.7的环境。

2.安装

在/home/shalom/anaconda3/envs路径下安装了python2的虚拟环境：

conda create -n python2 python=2.7

3.切换

命令行模式下，从默认python3切换到python2：

conda activate python2

此时，可以通过conda install xxx和pip install xxx来安装python2的相关工具包。

4.退出

退出python2环境：

conda deactivate

5.末了的话

类似地，可以安装任何虚拟的python版本，只要把名字一一对应。
