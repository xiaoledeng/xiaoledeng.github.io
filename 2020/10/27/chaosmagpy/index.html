ChaosMagPy: Evaluate the CHAOS geomagnetic field model

2020-10-27
#Researchsoftware #Python

1. Introduction

ChaosMagPy is a simple Python package for evaluating the [CHAOS-7](http://www.spacecenter.dk/files/magnetic-models/CHAOS-7/) geomagnetic field model.

Install:

pip install chaosmagpy

Documentation: ChaosMagPy documentation — ChaosMagPy 0.4 documentation
