霜降：秋天的最后一次回眸

2020-10-23
#Season #Life

1.简介

霜降（First Frost），是二十四节气中的第18个节气，是秋季的最后一个节气，在每年公历10月23—24日交节，是秋季到冬季的过渡。

元·吴澄《月令七十二候集解》说：“九月中，气肃而凝，露结为霜矣”。

“秋三月，此谓容平。天气以急，地气以明。早卧早起，与鸡俱兴。使志安宁，以缓秋刑。收敛神气，使秋气平。毋外其志，使肺气清，此秋气之应，养收之道也，逆之则伤肺，冬为飧泄，奉藏者少。” ——《黄帝内经素问.四气调神大论篇》

2.起居

寒霜轻轻降，冬阳徐徐近，出门加衣裳，感冒要预防，饮食要调节，心情要舒畅。

懂节制：霜降节气养生，需助阳气敛藏，同时适当温润补益身体，可借金气肃降之大势，将阳热一并潜入体内，刚好固护肾精，温煦肾水，待来年温木生发。

  1. 一定要注意防寒保暖，四肢和颈项、后背切不可受寒；
  2. 起居当早卧晚起，保证充足睡眠；
  3. 早起梳头，晚上泡脚；
  4. 趁天气好的时候，多出去晒晒太阳。
  5. 要减少户外运动，特别是剧烈运动，因为我们要准备进入冬天的静养阶段了。同时，雾霾天也会增多，根据自己的条件选择合适的空气净化设备，多吃一些润肺的食物。
  6. 还要特别注意不要熬夜，熬夜会使肝血不足，夜里更难入睡，形成恶性循环，还容易引起脾气暴躁、上火、皮肤干燥和便秘等。

护好腰：霜降一过百草枯，保腰护腿要知足。

  1. 穿高腰裤令腰温暖；
  2. 倒着走和后抬腿的动作可以锻炼背部肌群；
  3. 泡脚的时候以搓热的双手摩搓后腰；
  4. 每小时起身伸懒腰、拉筋。

3.饮食

贴秋膘：冬补不如补霜降。

防秋燥：多食健脾养阴润燥的食物，如柚子、萝卜、栗子、秋梨、百合、银耳、蜂蜜等食物。

饮食以生津养阴、益气固表、润燥祛痰为宜，花生莲藕排骨汤，玉米胡萝卜排骨汤，黄豆炖猪蹄，红枣枸杞煲乌鸡，参鸡汤，银耳汤等都可以常常喝起来。

适合霜降节气的食物还有：

  1. 山药，霜降后一定要多吃山药，山药的药用价值很高，其内富含蛋白质、糖类、淀粉和钙质，不仅营养高，而且热量低，饱腹感强，具有补脾养肺益肾的功效。
  2. 南瓜，南瓜营养价值高，易消化，能够改善我们的秋燥症状，是维生素A的重要来源，多吃还可以起到美容养颜的作用。
  3. 秋梨，梨味甜，性甘凉，能软化血管在秋季干燥时期很适合食用，想要清热润肺、生津降火的话，秋梨是一个不错的选择。
  4. 板栗，板栗是营养价值很丰富的一项食物，能够预防很多种疾病，所含的矿物质也很全面，富含维生素C，具有很强的滋补功效，霜降滋补再适合不过。
  5. 红薯，红薯是通便排毒的不错的食物，蛋白质含量高，能够健脾胃，强肾阴，除此之外，还能减肥养颜，防止亚健康。
