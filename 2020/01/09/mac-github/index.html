Mac设置Git上传本地项目到github

2020-01-09
#Experiences #Blog #Unix

  • 初始化当前目录：git init

  • 生成本地ssh (邮箱替换成你登录github的邮箱)：ssh-keygen -t rsa -C “youremail@youremail.com”

  • 将 ssh 代码复制到剪贴板：pbcopy < ~/.ssh/id_rsa.pub

  • 配置github的ssh key：Github-settings-配置 ssh-new SSH key

  • 打开终端，验证一下是否添加ssh成功了: ssh -T git@github.com

  • 设置github 登录名以及登陆邮箱：

    ​ git config –global user.name “your name”

    ​ git config –global user.email “youremail@youremail.com”

  • 上传本地项目：

    ​ git init ​ git add -A ​ git commit -m “good”

  • 推送到github的xiaoledeng.github.io 只能使用 master分支

    ​ git push -f git@github.com:xiaoledeng/xiaoledeng.github.io.git master
