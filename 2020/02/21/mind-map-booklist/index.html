思维导图—书单

2020-02-21
#Experiences #Mindmap #Booklist

1. 五星 (★★★★★)

  • 《思维导图完整手册-东尼·博赞》2018：东尼·博赞是思维导图的原创者，推荐首次阅读。

  • 《看完就用的思维导图-刘艳》2019：作者是思维导图原创者东尼·博赞所认可的，有非常详实的实践。

  • 《21天学会思维导图-尹丽芳》2019：作者是文科生，手绘能力强，而且是思维导图原创者东尼·博赞所认可的。

  • 《思维导图高效记忆古诗词-张维 谢庆平》2019：思维导图适合用来记忆小学的75首诗词。

  • 《思维导图宝典：好看又好用的导图大全集-吴帝德》2017：作者的话——我坚信，这应该是一本分量十足的关于思维导图的书，我以自己的亲身经历，将能设想到的大部分构思都通过案例的方式来展现给各位读者，通过文字讲述了我画每一幅思维导图的思考过程，我想，这应该是市面上相关书籍里独一无二的。

2. 三星 (★★★)

  • 胡雅茹

    • 《我的第一本思维导图入门书》2014
    • 《思维导图：创意高手的超强思考工具》2016
    • 《思维导图阅读法》2016
    • 《我的第一本思维导图入门书（精装版）》2017：比较适合学生来实践。
    • 《思维导图笔记整理术》2018：比较不错的笔记整理入门和实践。

  • 《零基础入门学习思维导图：提升个人效能-温大阿钱》2019：一个大学辅导员的零基础入门思维导图的指导书。另外，Xmind有详细介绍。

  • 《哈佛气质修炼手册（思维导图实战版）-聂凌风》2014：关于独立、自信、大气、智慧、细腻、沉稳、负责、坚强、乐观和淡定十方面的具体训练建议，思维导图凝练的不错！

  • 孙易新

    • 《零基础思维导图法》2017
    • 《学一次用一辈子的思维导图》2018
    • 《思维导图应用宝典》2015
    • 《思维导图：快速提升写作力等》2017
    • 《小学生思维导图作文课》2019
    • 《思维导图：提升你的职场核心竞争力》2017

  • 《词汇：新日语能力考试思维导图训练法-管静 吴帝德》2018：日语词汇进行思维导图记忆。

  • 《思维导图：职场高效能人士实用工具-贺菊中》2018

  • 《别说你懂思维导图-赵国庆》2015：理论比较强，每一章都有参考文献。

  • 《七天学会思维导图-石伟华》2019：适合引导小学生动手画思维导图。

3. 一星 (★)

  • 白虹：用imindmap绘制的思维导图类大全。

    • 《思维风暴（彩图精装）》 2013
    • 《思维导图（彩图精装）》2014
    • 《你一学就会的思维导图》2018

  • 《思维导图大全集-李猛》2010

  • 《思维导图大全集（彩图版）-李猛 龙潇》2012

  • 《你一学就会的思维导图-赵胤丞》2019

  • 《思维导图：走向高效的教与学-秦红斌》2019

  • 《思维导图学与用-思学团队》2019

  • 《思维导图攻略：快速上手与落地实践-王健文》2019

  • 《思维导图的三招十八式-张鄂勇》2012

  • 《思维导图从入门到应用-胡涵林》2019

  • 《思考的技术：解决关键问题的思维导图-王晓》2015

  • 《思维导图应用魔方：商务问题终结者-蔡劲松》2015

  • 《思维导图：超好用的时间管理法-姑苏明月》2017：故事的形式展示。

  • 《思维导图应用实战：画出你的时间表-格扎维埃·德伦盖尼》2019

  • 《日常生活中的思维导图-矢岛美由希》2016

4. 思维、记忆方面

  • 《思维风暴大全集10种黄金思维+思维导图+700个思维游戏+500道思维名题-张荣华》

  • 《最强大脑：快速记忆·逻辑思维·思维导图训练手册-朱建国 桑楚》2015

  • 《超级脑力训练：高效记忆、快速阅读与思维导图-雷南燕》2018：可以作为记忆方面的参考书。

  • 《超强大脑训练：超级记忆、思维导图与快速阅读-黄玉强》2017：可以作为记忆方面的参考书。

  • 《超实用记忆力训练法-吴帝德》2016

  • 《结构思考力：用思维导图来规划你的学习与生活 慕课版-曾啸波 李忠秋》2017
