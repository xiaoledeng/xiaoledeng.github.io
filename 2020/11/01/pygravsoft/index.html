1. Introduction

pyGravsoft is a Python interface to GRAVSOFT gravity field programs.

GRAVSOFT is a package of FORTRAN programs for gravity field modelling. The specific use of the programs for the determination of the empirical covariance function and its analytic modelling and geoid determination using least-squares collocation is described.
