Ubuntu安装向日葵远程控制

2020-11-13
#Unix #Experiences #Ubuntu

1. 下载

下载页面： https://sunlogin.oray.com/download/

分为命令行和图像版。

2. 安装命令

sudo dpkg -i sunloginclientshell-10.1.1.28779-amd64.deb

3. 安装问题解决

报错libwebkitgtk-3.0-0 is not installed，缺少依赖问题。

sudo gedit  /etc/apt/source.list 
#然后复制 deb http://cz.archive.ubuntu.com/ubuntu bionic main universe 到文件
sudo apt-get update
sudo apt-get install -f

4. 打开和绑定向日葵

通过命令行开启向日葵:

sudo /usr/local/sunlogin/bin/sunloginclient
  • 开启程序后的初始状态为未绑定，可见界面左上角的提示“Sunlogin （F12）”为进入菜单选项
  • 敲击F12进入菜单并通过键盘↑↓键选择Bind进行绑定账号
  • 入账号和密码，通过↓键移至ok处后敲击回车进行登录
  • 获得Linux主机的识别码(Code)和验证码(Access)

5. 卸载命令

sudo dpkg -l | grep sunlogin
sudo dpkg -r sunloginclient
