SLURM任务系统简要说明

2020-11-16
#Unix #Experiences

1. 基本命令

1.1 显示队列

# 显示队列
sinfo 

# 显示队列详细信息
scontrol show partition <partition name>

1.2 作业状态

squeue

# 查询指定用户名的作业状态
squeue -u <user_name>

1.3 显示节点

scontrol show node

1.4 取消任务

scancel jobid
