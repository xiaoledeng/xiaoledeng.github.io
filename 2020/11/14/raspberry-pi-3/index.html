Raspberry Pi【3】操作系统清单

2020-11-14
#Unix #Lists

1. 整理版

  • Raspberry Pi OS : 默认的最常用于Raspberry Pi的免费操作系统
  • Pidora : 专为树莓派优化的 Fedora Linux 系统
  • Arch Linux ARM
  • Ubuntu MATE : 是 Ubuntu Linux 官方的一个派生版，基于桌面环境 MATE
  • Ubuntu Desktop
  • Ubuntu Server : Ubuntu 专门用作服务器的版本
  • Ubuntu Core : 流行的 Ubuntu Linux 操作系统的精简版，旨在安装在各种嵌入式设备上
  • LibreELEC : 免费开源的轻量级“Just enough OS”Linux 发行版，它属于 KODI 的一个分支版本，功能类似于 Emby 和 PLEX 的多媒体服务平台。不同之处在于专为嵌入式设备而设计，支持各种 TV-BOX HTPC 和 RaspberryPi
  • Windows 10
  • OSMC : 使得树莓派更适合做个xbmc播放器，是 Openelec 的升级版，同样是基于 Kodi 的开源项目
  • Openelec : Open Embedded Linux Entertainment Center，开源嵌入式Linux 娱乐中心
  • PiNet : 帮助学校建立和管理一个Raspberry Pi的课堂
  • Mozilla WebThings : Mozilla 的物联网平台项目
  • weather_station-2018-01-14 : 基于树莓派全自动气象站，可应用多种传感器
  • RISC OS : 第一款ARM计算机的操作系统
  • Kali Linux : 高级渗透测试和安全审计Linux发行版
  • NOOBS : 从安装的多个树莓派系统中选择你要启动的系统
  • Sabayon : 在5分钟以内使一台电脑进入强大的Gentoo Linux系统
  • Recalbox : 复古模拟平台
  • CentOS
  • DietPi : 基于 Raspbian 的精简版本系统，其特点就是小和快
  • SUSE Linux
  • Lakka : RetroArch 官方的 Linux 发行版，以多机种游戏模拟器 libretro 为核心
  • RetroPie : 将复古游戏机集成到Raspberry pi 树莓派上使之成为一个专用的游戏机模拟器平台
  • Kano : 专门面向儿童编程量身定做的树莓派镜像，让儿童在游戏中学习编程
  • RTandroid : 定制版的安卓系统，可以正常安装安卓软件，wifi蓝牙工作正常
  • Flintos : 一款更快速，更简单，更高效的云驱动操作系统

2. MAKE 趣无尽

  • Puppy Linux - 面向家庭用户的一款 Linux 发行版
  • MoodeAudio - 出色的树莓派发烧级质量音乐播放系统
  • Raspberry Pi OS（Raspbian）- 树莓派官方系统
  • RasPlex - 树莓派的 Plex 客户端
  • RuneAudio - HiFi 音乐播放器
  • XBian - 一个小型、轻巧的媒体中心 Linux 发行版
  • openSUSE - 一个开源的 Linux 发行版
  • OpenMediaVault - 基于 Debian 的 NAS 包含 SSH、FTP、SMB/CIFS 等服务
  • OpenBSD - 免费的类 Unix 操作系统
  • NetBSD - 免费、快速、安全且高度可移植的类 Unix 开源操作系统
  • Manjaro - 树莓派的友好、开源 Linux 发行版
  • motionEyeOS - 基于 Linux 的视频监视系统
  • Hass.io - 嵌入式设备的家庭自动化操作系统/应用程序
  • Gladys Assistant - Gladys，隐私优先的开源家庭助理
  • Gentoo - 树莓派的 Gentoo Stage 3 压缩包
  • FreeBSD - FreeBSD/ARM on the Raspberry Pi family
  • 为树莓派构建的 Linux Fedora 发行版
  • DietPi - 设计成可容纳2GB SD卡的最小图像，具有大量可配置的设置和脚本
  • Debian-Pi-Aarch64
  • Chromium OS - Chromium OS for Raspberry Pi
  • balenaOS - 用于在嵌入式设备上运行 Docker 容器的开源 OS
  • Alpine Linux - 基于 musl libc、busybox 的轻量级 Linux 发行版
  • OpenELEC - 运行快、且用户体验友好的一款XBMC媒体中心
  • Kodi - 家庭多媒体影音中心播放器
  • Android Things - 构建适用于各种消费，零售和工业应用的连接设备
  • RetroArch
  • EmulationStation
  • OctoPrint - 用于 3D 打印机的发行版
  • Home Assistant - 智能家居平台系统
  • 给树莓派安装完整版 Windows 10 的工具 - 趣小组
  • Kali Linux - 极客喜爱的 Linux 发行版
  • OpenWrt - 一个嵌入式的 Linux 发行版
  • RetroPie - 经典游戏模拟器系统
  • Recalbox - 游戏模拟器系统和媒体中心
  • Volumio 音乐播放器系统
  • CentOS ARM - CentOS ARM 下载
  • Arch Linux ARM - 轻巧灵活的 Linux 发行版，力求使其简单
  • RISC OS - 用于 ARM 设备的操作系统
  • PiNet - 用于设置和管理教室里树莓派的集中式用户帐户和文件存储系统
  • Mozilla WebThings - 智能家居网关系统
  • LibreELEC - 精简化的 Kodi 系统
  • OSMC 多媒体盒子系统
  • Windows10 IoT - 微软的物联网系统
  • Ubuntu Server - 树莓派的 Ubuntu 服务器发行版
  • Ubuntu Core - 树莓派的 Ubuntu 核心发行版
  • Ubuntu MATE for the Raspberry Pi
  • RT-Thread, RTOS, 物联网操作系统 - RT-Thread物联网操作系统
  • RetrOrangePi
