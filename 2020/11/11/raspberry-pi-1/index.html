Raspberry Pi【1】树莓派 (Raspberry Pi, RPi)

2020-11-11
#Unix

Raspberry Pi (中文名为“树莓派”，简写为RPi，是为学习计算机编程教育而设计)，只有信用卡大小的微型电脑，其系统基于Linux。

The Raspberry Pi is a tiny and affordable computer that you can use to learn programming through fun, practical projects.

官网： https://www.raspberrypi.org/

Documentation: https://www.raspberrypi.org/documentation/

Download: https://www.raspberrypi.org/downloads/
