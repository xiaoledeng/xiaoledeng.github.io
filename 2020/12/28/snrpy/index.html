1. Introduction

开源的 GNSS 干涉反射测量软件包 SNRpy，翻译自摘要：

全球导航卫星系统(GNSS)干涉反射法，也被称为GNSS-IR，使用来自大地测量质量的GNSS天线的数据来提取天线周围环境的信息。土壤湿度监测是GNSS-IR技术的重要应用之一。这手稿展示的主要思想和实现决策需要编写的Python代码的软件工具，观察和导航文件RINEX格式转换成适当的格式GNSS-IR(包括信噪比观测和卫星的方位角和仰角)并确定反射高度和干涉波的相位和振幅值调整为每个单独的卫星轨道。该手稿的主要目标是与科学界分享该软件，向新用户介绍GNSS-IR技术。

2. Reference

Martín, A., Luján, R. & Anquela, A.B. Python software tools for GNSS interferometric reflectometry (GNSS-IR). GPS Solution, 24, 94 (2020). https://doi.org/10.1007/s10291-020-01010-0
