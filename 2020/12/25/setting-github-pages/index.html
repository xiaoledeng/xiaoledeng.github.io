Github Pages的设置

2020-12-25
#Blog

1. 问题描述

Github Pages的个人主页出现404的错误，即无法访问。

2. 问题原因

Github教育用户到期，Github Pages的Private设置导致的，将Private改为Public即可。

3. 解决方法

打开Github Pages的项目地址，点击Settings，找到GitHub Pages设置，而后设置为Public，并选好Branch (Main 或 Master) 和博客地址，直到显示为：

Your site is published at https://www.xiaoledeng.com/

问题得到解决，个人主页再次可以访问。
