Word调整脚注分隔线

2025-02-21
#Unix #Windows #Experiences

脚注分隔线【Footnote Separator】 的格式会导致空白增大。

  1. 切换到 草稿 视图：

    • 点击 视图 View → 选择 草稿 Draft。

  2. 显示脚注分隔线：

    • 点击 引用 References → 显示注释 Show Notes。
    • 在脚注区域顶部的下拉框中，选择 脚注分隔线 Footnote Separator。

  3. 编辑分隔线：

    • 删除分隔线上的空行，或将分隔线缩短。
    • 如果需要，可以重新绘制简短的分隔线。

  4. 返回页面视图：

    • 点击 视图 View → 选择 页面视图 Print Layout 检查效果。
