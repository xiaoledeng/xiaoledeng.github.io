Ubuntu安装Tenda AX300无线网卡

2025-02-14
#Ubuntu #Experiences

1. 引言

在 Ubuntu 的台式机上，因为房间没有网线接口，需要使用 Wi-Fi 联网。

台式机上的原装 Wi-Fi 天线找不到，故利用 USB 的无线网卡。【注：后面原装天线找到，比 USB 无线网卡好用。】

2. 步骤

  1. 在 W311MIV6 国产银河麒麟系统（Kylin）及Linux（Ubuntu）驱动（支持内核3.10~6.8） 下载驱动压缩包，解压；

  2. 在终端输入： sudo dpkg -i ax300-wifi-adapter-linux-driver.deb，直到提示安装成功；

  3. 将无线网卡插入台式机；

  4. 执行命令 sudo rfkill unblock wifi，解决无线网卡 Wi-Fi 模块异常情况。

注意⚠️：更新系统后，需要重新安装驱动。
