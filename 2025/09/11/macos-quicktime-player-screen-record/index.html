macOS利用QuickTime Player屏幕录制

2025-09-11
#Unix #Experiences #macOS

1. 引言

在 macOS 系统中，不需要额外安装第三方软件，就可以通过系统自带的 QuickTime Player 来完成屏幕录制。

无论是制作演示教程、保存操作过程，还是录制讲解视频，QuickTime 都能满足基础需求。

2. 录制步骤

  1. 打开 QuickTime Player
  • 路径：应用程序 (Applications) → QuickTime Player
  • 也可以用 Spotlight 搜索 (Spotlight Search)：按 Command + 空格 输入 QuickTime Player。
  1. 新建屏幕录制
  • 菜单栏选择：文件 (File) → 新建屏幕录制 (New Screen Recording)
  • 会出现一个录制工具条。
  1. 设置录制选项
  • 点击 红色圆点 (Record Button) 开始录制。
  • 系统会提示：
    • 录制整个屏幕 (Record Entire Screen)
    • 录制部分区域 (Record Selected Portion)
  • 在 选项 (Options) 里，可选择：
    • 是否录制 麦克风 (Microphone)
    • 视频保存位置 保存路径 (Save Location)
  1. 开始录制
  • 选择录制范围后，点击 开始录制 (Start Recording)。
  • 此时屏幕右上角菜单栏会出现一个 停止按钮 (Stop Button)。
  1. 停止录制
  • 方法一：点击菜单栏的 停止按钮 (Stop Button) ⏹️
  • 方法二：按快捷键 Command + Control + Esc
  1. 保存视频
  • 录制结束后，QuickTime 会自动弹出视频预览窗口。
  • 菜单栏选择：文件 (File) → 保存 (Save)
  • 输入文件名，选择 保存位置 (Save Location) → 点击 存储 (Save) 即可。

3. 总结

通过 QuickTime Player，可以快速完成屏幕录制，操作简洁直观，足以满足日常工作、学习和演示的需求。
