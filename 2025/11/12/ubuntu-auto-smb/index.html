Ubuntu自动挂载SMB

2025-11-12
#Unix #Ubuntu

1. 引言

每次连接 SMB 文件，都需要输入 root 密码。

解决方案是：让共享在用户目录下自动挂载，不再需要 root。

2. 方法

1、安装必要工具：

sudo apt install cifs-utils gvfs-backends gvfs-fuse

2、使用文件管理器访问 SMB，登录一次后，缓存密码（除非关闭了「记住密码」）：

smb://192.188.1.110/share

3、打开启动应用程序设置：

  • 在应用程序菜单中搜索 Startup Applications或启动应用程序
  • 或按 Super 键，输入startup

4、添加信息：

  • 名称：Auto SMB
  • 命令：gio mount smb://username@192.188.1.110/share
  • 注释：Auto SMB
