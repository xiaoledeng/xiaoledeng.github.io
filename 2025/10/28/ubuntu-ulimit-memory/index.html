Ubuntu 24.04 LTS 解开默认内存限制

2025-10-28
#Unix #Ubuntu

1. 引言

Ubuntu 24.04 LTS 系统默认物理内存有约16G的限制，需设置成无限制。

2. 步骤

1、查看系统内存是否有限制：

ulimit -a | grep "max locked memory"

# 这是唯一有限制的内存相关项，约 16 GB
max locked memory           (kbytes, -l) 16447196

2、解除限制操作

修改 sudo vim /etc/security/limits.conf：

*    soft    memlock    unlimited
*    hard    memlock    unlimited

修改 limits.conf 后，需注销并重新登录。

3、检查是否解除限制：

ulimit -a | grep "max locked memory"

# 显示内容如下即可
max locked memory           (kbytes, -l) unlimited
