macOS利用VS Code的Remote SSH连接Ubuntu

2025-10-22
#Unix #macOS #Ubuntu #Software

1. 引言

当在 macOS 上写代码、在 Ubuntu 上运行时，容易出现各种兼容问题，比如 ._ 隐藏文件。

解决方案之一是 VS Code的 Remote SSH 模式。

2. 设置

安装插件：

  • 打开 VS Code → 左侧扩展（或快捷键 ⇧⌘X）
  • 搜索并安装 Remote - SSH（微软官方发布）

添加远程主机：

  • 按 ⇧⌘P → 输入：Remote-SSH: Connect to Host…
  • 选择 “Add New SSH Host”，输入例如：ssh user@192.168.1.180
  • 保存后重新连接。

打开远程项目：

  • 连接成功后，左下角会显示 >< SSH: your-server

此时，打开的文件夹、终端、编译都在远程 Ubuntu 上完成。

3. 优点

  • 无需拷贝文件，直接在远程运行
  • 不会生成 ._ 文件
  • 支持远程终端、调试、Git 管理
  • 环境保持一致，省心又高效

4. 总结

用 VS Code的 Remote SSH 让 macOS 变成远程 Ubuntu 的开发窗口，再不担心文件污染或环境不一致。
