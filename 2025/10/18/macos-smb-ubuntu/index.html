macOS通过SMB/Samba直接访问Ubuntu文件

2025-10-18
#Unix #macOS #Ubuntu

1. 在 Ubuntu 上设置

# 1. 安装 Samba
sudo apt update
sudo apt install samba

# 2. 创建共享目录或共享现有目录
sudo smbpasswd -a your_username  # 设置 Samba 密码

# 3. 编辑配置文件
sudo nano /etc/samba/smb.conf
# 添加共享配置：
[Name]
   path = /home/your_username
   browseable = yes
   read only = no
   valid users = your_username

# 4. 重启 Samba：
bashsudo systemctl restart smbd

2. 在 macOS 上连接：

1、打开 Finder

2、 按 Cmd + K 或选择前往 > 连接服务器

3、输入：smb://ubuntu的IP地址

4、输入 Ubuntu 的用户名和 Samba 密码
