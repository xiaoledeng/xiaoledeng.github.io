yt-dlp保持最新版本

2025-10-24
#Unix #macOS #Software

1. 引言

yt-dlp 是一个强大的命令行工具，用于从YouTube和其他视频网站下载视频。

核心特点：

  • 广泛支持：支持1000+视频网站(YouTube、B站、Twitter等)，见：yt-dlp支持的部分中文网站
  • 高性能：比原版youtube-dl更快，功能更多
  • 格式灵活：可选择视频质量、格式，支持音频提取
  • 活跃维护：社区持续更新，快速修复问题

yt-dlp 需要保持最新版本，不然下载视频等会有各种稀奇古怪的错误。

2. 保持最新版本的macOS命令

虽然 macOS 可以通过 brew install yt-dlp 和 brew upgrade yt-dlp，但还是赶不上 GitHub 上的更新。

通过以下命令，下载 GitHub 最新的版本：

curl -L https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp -o ~/yt-dlp && \
chmod +x ~/yt-dlp && \
yt-dlp --version

在 vim ~/.zshrc 加入：export PATH="$HOME:$PATH"，即可使用 yt-dlp 命令。
