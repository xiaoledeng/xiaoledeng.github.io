Ubuntu 22.04 LTS 命令行升级到 24.04 LTS

2025-10-16
#Unix #Ubuntu 
# 1. 备份重要数据（强烈建议）

# 2. 更新当前系统
sudo apt update
sudo apt upgrade -y
sudo apt dist-upgrade -y

# 3. 移除不需要的包
sudo apt autoremove -y
sudo apt autoclean

# 4. 确保 update-manager-core 已安装
sudo apt install update-manager-core -y

# 5. 检查升级配置（可选）
# 编辑 /etc/update-manager/release-upgrades
# 确保 Prompt=lts（这是默认设置）
sudo vim /etc/update-manager/release-upgrades

# 6. 开始升级过程
sudo do-release-upgrade

# 7. 重启系统
sudo reboot

# 8. 验证升级
lsb_release -a
# 显示以下内容
# No LSB modules are available.

# Description:	Ubuntu 24.04.3 LTS
# Release:	24.04
