macOS系统添加 RICOH 打印机

2025-04-04
#Unix #Experiences #macOS

  1. 确认打印机当前 IP 地址

  2. 确保 Mac电脑 和打印机在同一网络

  3. 安装打印机驱动，比如：RICOH IM C3010 PS 的驱动下载地址为：IM C3010/C3510 Downloads | Ricoh Global

  4. 打开「系统设置 System Settings」—「打印机 Printers & Scanners」—「添加打印机 Add Printer, Scanner or Fax」—「输入IP地址」—「选择协议：Internet Printing Protocol 或 Line Printer Daemon」—「Use 选择RICOH IM C3010 PS」—添加完成即可。

最后一步，如果打印机没有响应，问题很可能出在协议方面：Internet Printing Protocol 或 Line Printer Daemon 或 HP Jetdirect - Socket。
