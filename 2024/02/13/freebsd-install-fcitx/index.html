FreeBSD 14.0 安装中文Fcitx输入法

2024-02-13
#Unix #FreeBSD #Unix

FreeBSD 14.0，具体版本为：14.0-RELEASE-p3。

1. 安装中文字体

pkg install zh-CJKUnifonts

2. 安装并设置 Fcitx 输入法

安装命令：

pkg install zh-fcitx zh-fcitx-configtool zh-fcitx-libpinyin

在 ~/.zshrc加入以下命令：

export XMODIFIERS='@im=fcitx'
export GTK_IM_MODULE=fcitx
export GTK3_IM_MODULE=fcitx

执行以下命令：

mkdir -p ~/.config/autostart
cp /usr/local/share/applications/fcitx.desktop ~/.config/autostart/

3. 设置 Fcitx 输入法为小鹤双拼

搜索关键词 xiaohe：

cd ~/.config/fcitx
grep -ir xiaohe

./conf/fcitx-libpinyin.config:# XiaoHe

打开文件：

vim ~/.config/fcitx/conf/fcitx-libpinyin.config

设置双拼方案为小鹤：

[Shuangpin]
# Shuangpin Scheme
# Available Value:
# Ziranma
# Microsoft
# ZiGuang
# ABC
# PinyinJiaJia
# XiaoHe
#Scheme=Ziranma
Scheme=XiaoHe

点击右上角的输入法按钮，重启，选择双拼输入法即可。
