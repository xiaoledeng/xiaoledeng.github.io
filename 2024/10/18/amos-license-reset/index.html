Windows系统Amos证书过期重置

2024-10-18
#Windows #Software

1. 问题

Amos证书过期信息如下：

License Validation

The temporary period for running Amos without a license has expired. Use the License Authorization Wizard to contact SPSS for a license code.

2. 解决步骤

  1. 删除 C:\ProgramData\SafeNet Sentinel\Sentinel RMS Development Kit\System\ 目录下的所有文件

  2. 在Amos的默认安装目录 C:\Program Files (x86)\IBM\SPSS\Amos\26 下找到 licenseinit.exe 文件，用管理员身份运行

  3. 运行后，弹出对话框，第一个是 Base product feature code:，填 1，回车；第二个是 Version (with a decimal point):，填 26（因Amos版本号的是26），回车

  4. 重启Amos即可
