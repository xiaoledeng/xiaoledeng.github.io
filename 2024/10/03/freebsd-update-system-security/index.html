FreeBSD更新系统和安全漏洞

2024-10-03
#FreeBSD #Unix

在 FreeBSD 的安全页面FreeBSD Security Advisories | The FreeBSD Project，有定期发布的安全漏洞，需要及时对系统进行更新。

  1. 查看系统版本：
freebsd-version

显示：

14.0-RELEASE-p5
  1. 更新系统命令：

注意⚠️：管理员 root 操作

freebsd-update fetch
freebsd-update install

有以下提醒：

WARNING: FreeBSD 14.0-RELEASE-p5 HAS PASSED ITS END-OF-LIFE DATE.
Any security issues discovered after Tue Oct 1 08:00:00 CST 2024
will not have been corrected.

FreeBSD Security Information | The FreeBSD Project显示：

FreeBSD 14.0-RELEASE 系统的维护起止日期为：November 20, 2023 - September 30, 2024

FreeBSD 14.1-RELEASE 系统的维护起止日期为：June 4, 2024 - March 31, 2025

  1. 重启：
shutdown -r now
shutdown -r +10min "Rebooting for a security update"
  1. 查看系统版本：
freebsd-version

显示：

14.0-RELEASE-p11
