Python去除指定字符strip和replace

2024-01-03
#Python

1. 引言

Python 去除字符串的指定字符时， strip 和 replace 的区别：

  • strip 去掉首尾的字符；默认空的话，去除空格
  • replace 去掉中间任意字符

2. 示例

a = '2023-01-03'
b = a.strip('-')
print(b)
c = a.replace('-','')
print(c)

结果：

2023-01-03
20230103
