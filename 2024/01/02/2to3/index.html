2to3: Python2代码转换为Python3

2024-01-02
#Python

1. 前言

之前写的 Python 2 代码，可以通过官方自带工具 2to3 1 自动转换为 Python 3 代码。

经过测试， Python 3.9.13 可以用 2to3 工具。

2. 用法

2to3 -n -W --add-suffix=3 example.py

会生成 example.py3 。

说明：

  • -n 将备份原始文件
  • -w 标志启用回写回更改
  • -l 标志列出了所有可用的修复程序
  • -f 给出要运行的显式修复程序集
  • -x 明确禁用了修复程序
  • -v 可以输出有关翻译过程的更多信息。
