FreeBSD对Linux的支持

2024-01-19
#Unix #FreeBSD #Unix

1. FreeBSD 配置 Linux 二进制兼容性

sysrc linux_enable="YES"

service linux start

2. CentOS 支持

pkg install linux_base-c7

此时，/compat/linux 路径下是 CentOS 7 的基础系统文件。

3. Debian / Ubuntu 支持

pkg install debootstrap
debootstrap focal /compat/ubuntu

此时，/compat/ubuntu 路径下是 Ubuntu 的基础系统文件。
