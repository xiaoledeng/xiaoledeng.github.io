macOS安装SageMath

2024-01-01
#Unix #Python #macOS #Software

1. 引言

根据安装说明 1，macOS 有三种安装方式：

  1. dmg 文件
  2. conda-forge
  3. source 源码 2

2. 安装

dmg 文件安装最简单：

  1. 下载 dmg 文件：Releases · 3-manifolds/Sage_macOS，M 芯片选 arm64；Intel 芯片选 x86_64。10.2版本的文件大约有 1.02 GB。

  2. 双击 dmg 文件，移动 app 文件到 Applications；安装 pkg 文件。

3. 使用

  • 终端输入： sage ，进入命令行模式。

  • 点击 SageMath 图标，选择进入 命令行 或 Notebook

具体使用，参考使用手册 3 (Tutorial: The best way to become familiar with Sage in only a few hours)。
