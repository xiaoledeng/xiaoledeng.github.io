FreeBSD设置默认编辑器

2024-01-06
#Unix #FreeBSD #Unix

在 ~/.shrc 和 ~/.zshrc 添加以下内容：

alias vi="vim"
export EDITOR=/usr/local/bin/vim

source ~/.shrc
source ~/.zshrc

这样，使用 vi/ vim/ crontab -e 命令，都能用 vim 编辑器。
