Ubuntu的不足和优点

2024-01-12
#Unix #Ubuntu

1. 不足

  1. 强制广告：在命令行强制为 Ubuntu Pro 订阅服务做广告，服从公司利益1, 2, 3, 15

Receive additionaL future security Updates with Ubuntu Pro. Learn more about Ubuntu Pro at https://ubuntu.com/pro

通过 Ubuntu Pro 接收额外的未来安全更新。了解更多关于 Ubuntu Pro 的信息，请访问 https://ubuntu.com/pro

  1. 无隐私：搜索结果会发送给 Amazon 等，方便打广告。

在默认情况下，Canonical 会将加密发送到 Ubuntu 服务器的搜索关键词和 IP 信息储存起来，并转发给第三方合作伙伴，包括 Facebook, Twitter, BBC 和 Amazon。4, 5, 6

  1. 收集用户信息：默认是打开 Ubuntu one，收集用户的使用信息7

  2. 默认安装不精简：占资源，耗内存，开机默认启动进程太多8, 9

  3. Snap 软件包，速度慢和 bug 多，不属于开源软件10, 11, 12, 15

  4. apt 安装某软件时，软件包容易冲突（拆包细、乱改名、版本旧）13, 14

  5. 桌面版容易崩溃，有错误弹窗16

  6. 安全漏洞以更新整个系统的形式发布17

其他：新的 LTS 长期版发布后，旧的 LTS 长期版基本不更新，一般 5 年后，服务器裸奔或重装；跨版本升级可能升不动。

2. 优点

来自： 你为什么喜欢用 Ubuntu? - 知乎

  1. 系统安装简单，开箱即用，默认配置一般满足需求

  2. 驱动兼容性好

  3. 桌面体验好

  4. 一般是新手入门推荐的 Linux 第一个发行版

  5. Ubuntu 出了问题，网上搜索一般都有解决方案

  6. 一般的开源程序都会支持 Ubuntu
