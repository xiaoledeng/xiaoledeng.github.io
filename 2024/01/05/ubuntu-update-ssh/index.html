Ubuntu更新SSH

2024-01-05
#Unix #Ubuntu

查看 SSH 当前版本：

ssh -V
# 显示
OpenSSH_8.9p1 Ubuntu-3ubuntu0.5, OpenSSL 3.0.2 15 Mar 2022

查看 SSH 位置：

which ssh
# 显示
/usr/bin/ssh

更新 SSH 命令：

sudo apt update
sudo apt upgrade openssh-client
