macOS升级后usekeychain错误解决

2024-01-05
#Unix #macOS

macOS 版本从 13.6.2 升级到 13.6.3，算是小版本升级。

在终端用 SSH 连接服务器时，出现以下错误：

/Users/name/.ssh/config: line 5: Bad configuration option: usekeychain
/Users/name/.ssh/config: line 13: Bad configuration option: usekeychain
/Users/name/.ssh/config: terminating, 2 bad configuration options

解决方案：在 ～/.ssh/config 文件的最开始加入以下命令：

Host *
  AddKeysToAgent yes
  IgnoreUnknown UseKeychain
  UseKeychain yes
