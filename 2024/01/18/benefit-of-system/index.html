随笔—系统内置的优势

2024-01-18
#Ideas #Essay #Unix #Minimalism #macOS #Software

前段时间，卸载了 Royal RSX 软件。因为 macOS 系统内置终端的功能（ SSH 访问服务器和 SFTP 上传和下载文件），能够替代 Royal RSX 软件提供的功能。这是系统内置工具的优势——安全和简洁。

在遇到一些 SSH 的设置问题时，虽然在网上搜索了很多经验，但是大多都是流于表面，人云亦云，没有实质地解决问题。

后面发现：能够实质解决问题的方式是查看系统内置的手册，更直接更具体，即：

# 查看 OpenSSH SSH 服务端的设置手册
man sshd_config

# 查看 OpenSSH SSH 客户端的设置手册
man ssh_config

系统内置的手册，一般有详细的说明，能够实质地解决问题。类似的例子还有：

man ssh-add
man yt-dlp
man node

总之，用系统内置工具，少用第三方软件；看系统内置手册，直击问题核心。
