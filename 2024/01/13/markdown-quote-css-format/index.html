Markdown的引用CSS样式

2024-01-13
#Markdown #Blog

Markdown 的引用 (quote) 为：>，修改其样式如下：

首先，CSS 样式文件的路径为 /themes/simple-style/static/css/main.css

其次，加入以下代码：

blockquote {
    border-left: 4px solid #8fe5f7;
    padding: 0 15px;
    color: #000000;
}
  • 4px 为宽度
  • #8fe5f7 为浅蓝色
