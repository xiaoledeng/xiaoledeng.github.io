Clear Linux安装oneAPI

2024-12-11
#Unix #ClearLinux #Fortran #C #Experiences
  1. 下载两个 Linux 的 sh 文件：
  • Get the Intel® oneAPI Base Toolkit
  • Get Intel® oneAPI HPC Toolkit
  1. 预先安装 g++ 包：
swupd bundle-add c-basic
  1. 安装：
sh intel-oneapi-base-toolkit-2025.0.0.885_offline.sh
sh intel-oneapi-hpc-toolkit-2025.0.0.825_offline.sh

按照图形界面的指示，完成安装。

  1. 设置环境变量并验证安装成功

在 ~/.bashrc 文件加入：

source /opt/intel/oneapi/setvars.sh

source ~/.bashrc

验证安装成功：

ifx --version  # 检查 Fortran 编译器版本

ifx (IFX) 2025.0.0 20241008
