Clear Linux默认进入命令行模式

2024-12-13
#Unix #ClearLinux
  1. Clear Linux 默认可能使用图形化目标（graphical.target），需要将其切换为多用户目标（multi-user.target）：
sudo systemctl set-default multi-user.target
  1. 启动图形界面时使用 startx

确保 xinit 和桌面环境已安装：

sudo swupd bundle-add x11-server xinit

如果需要 GNOME 桌面环境，可以安装 GNOME 相关软件包：

sudo swupd bundle-add desktop

确保 ~/.xinitrc 文件存在并包含启动桌面环境的命令。例如，如果使用 GNOME 桌面环境：

echo "exec gnome-session" > ~/.xinitrc

对于桌面环境 XFCE 或 KDE ，可以相应更改为：

echo "exec startxfce4" > ~/.xinitrc
echo "exec startkde" > ~/.xinitrc

启动图形界面，终端输入：

startx
  1. 恢复到图形模式
sudo systemctl set-default graphical.target
