英文学位学历和成绩单证明

2024-11-28
#Experiences #Lists #Life

1. 毕业学校

原始文件是从毕业学校获取，一般是在「图书馆的档案馆」。

武汉大学提供线上的电子凭证： 武汉大学可信电子凭证

用身份证号登录，可以选择「本科」、「硕士」或「博士」类型。

2. 学信网

  • CDGDC Degree Verification Report (中国高等教育学位认证报告)
  • CHSI Verification Report for the China Higher Education Qualification Certificate (教育部学历证书电子注册备案表)
  • CHSI Verification Report for the China Higher Education Student’s Academic Transcript (中国高等学校学生成绩验证报告)

学位和学历翻译件：

  1. 实名注册和登录： 学信档案

  2. 在「在线验证报告」的「高等学历」和「学位」，绑定报告。

  3. 「出国报告发送」的「英文翻译」，「继续申请」，选择「翻译类型」，「中国高等教育学位在线验证报告翻译件 或 学位在线验证报告翻译件」和「学历证书电子注册备案表翻译件 或 教育部学历证书电子注册备案表翻译件」

  4. 30元一份，英文翻译。

成绩单翻译件：

  1. 从「学校图书馆的档案馆」获取「中文或英文的成绩单」

  2. 登录 出国教育背景信息服务

  3. 「申请认证」，按照「申请流程」填写，仔细阅读上传的注意事项，中文：150元/份，英文：150/份。
