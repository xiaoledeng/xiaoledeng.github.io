FreeBSD从 14.0-RELEASE 升级到 14.1-RELEASE

2024-11-25
#FreeBSD #Unix

注意⚠️：管理员 root 操作

  1. 查看系统版本：
freebsd-version

显示：

14.0-RELEASE-p11

有以下提醒：

WARNING: FreeBSD 14.0-RELEASE-p11 HAS PASSED ITS END-OF-LIFE DATE.
Any security issues discovered after Tue Oct 1 08:00:00 CST 2024
will not have been corrected.
  1. FreeBSD大版本更新，从 14.0-RELEASE 到 14.1-RELEASE 的命令：
freebsd-update -r 14.1-RELEASE upgrade

/usr/sbin/freebsd-update install
  1. 重启后，再次输入：
/usr/sbin/freebsd-update install

后台会有 root 进程安装系统更新。

  1. 更新完后，不用重启。查看系统版本：
freebsd-version

显示：

14.1-RELEASE-p5

故此，大升级系统版本成功。

注意：14.1-RELEASE 过期时间为 March 31, 2025，见： https://www.freebsd.org/security/#sup
