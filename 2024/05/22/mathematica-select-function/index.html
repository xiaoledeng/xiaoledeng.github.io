Mathematica的Select函数

2024-05-22
#Mathematica

Select[list,crit]

选取 list 中使得 crit[ei] 为 True 的所有元素 ei.

其中，crit 的选项有：

  • IntegerQ
  • EvenQ
  • OddQ
  • NumberQ
  • NumericQ
  • MachineNumberQ
  • TrueQ
  • BooleanQ
