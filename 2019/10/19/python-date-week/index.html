Python 给定日期返回属于当年第几周

2019-10-19
#Python

1、返回当前时间对应当年第几周：

import datetime
datetime.datetime.now().isocalendar()
(2019, 42, 6)　　

结果表示当前日期对应2019年的第42周的第6天，注：当前日期是2019年10月19日

2、给定日期 返回对应当年第几周：

import datetime
datetime.date(2019, 10, 19).isocalendar()
(2019, 42, 6)　　

即为给定2019年10月19日，返回该日期对应2019年的第42周的第6天
