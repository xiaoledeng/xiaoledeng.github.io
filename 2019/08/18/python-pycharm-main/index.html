Python的Pycharm的Main错误

2019-08-18
#Python

在Pycharm编辑器中，运行Python脚本时，出现 can’t find ‘main’ module 的错误。

原因是：没有选择*.py 文件。

具体设置：编辑运行环境——脚本路径填写完整，即为完整路径需要填到最后的文件名。
