Python列表去重

2019-08-19
#Python

Python对列表进行去重，比较简洁的代码如下：

#原始列表
a = ["John","John","John","Mark","David","David","Shalom","Shalom","Shalom"]

#先使用set，结果是一个无序不重复元素集
#再使用list()，把字符串str或元组转成数组
b = list(set(a))

#按照索引再次排序
b.sort(key=a.index)

#打印显示去重后的列表
print (b)

运行结果为：

[‘John’, ‘Mark’, ‘David’, ‘Shalom’]

如果把列表换成数字也是可以的：

  a = [1,1,1,0,0,0,3,3,3,7,7,5,5,5]

运行结果为：

[1, 0, 3, 7, 5]
