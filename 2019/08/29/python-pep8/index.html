Python遵循PEP-8样式指南

2019-08-29
#Python

Python Enhancement Proposal＃8，也称为PEP 8，是如何格式化Python代码的样式指南。

英文原网址为：https://www.python.org/dev/peps/pep-0008/

中文翻译网址依次为：

  • PEP8中文版 – Python编码风格指南
  • Python PEP-8编码风格指南中文版
  • python代码风格指南(PEP8中文版)

注意事项

  • Pylint工具（http://www.pylint.org/）是一种流行的Python源代码静态分析器。 Pylint提供PEP 8样式指南的自动执行，并检测Python程序中的许多其他类型的常见错误。
  • 编写Python代码时，请始终遵循PEP 8样式指南。
  • 与较大的Python社区共享共同风格有助于与他人协作。
  • 使用一致的样式可以在以后更轻松地修改自己的代码。
