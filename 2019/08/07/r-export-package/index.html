R语言的export包

2019-08-07
#R

安装

install.packages("export")
library(export)

导出为ppt，并在ppt中修改

graph2ppt(file="name.pptx", width=7, height=5)

其他文件格式

graph2doc导出到Word，还支持通过graph2office导出到LibreOffice。

  • graph2bitmap
  • graph2doc
  • graph2eps
  • graph2jpg
  • graph2office
  • graph2pdf
  • graph2png
  • graph2ppt
  • graph2svg
  • graph2tif
  • graph2vector

表格：

  • table2csv
  • table2csv2
  • table2doc
  • table2excel
  • table2html
  • table2office
  • table2ppt
  • table2spreadsheet
  • table2tex
