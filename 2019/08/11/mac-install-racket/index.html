Mac安装Racket并加入环境变量

2019-08-11
#Unix

安装Racket

Racket是一种通用编程语言，也是世界上第一个面向语言编程的生态系统。

Racket下载的地址为：Download Racket，包括了Mac、Windows和Linux等平台。

Mac下载，默认安装之后，安装路径为：/Applications/Racket 7.3

加入环境变量

在命令行下面输入Racket，发现没有加入环境变量。先打开：

$ vi ~/.zshrc

加入命令：

export PATH="/Applications/Racket/bin:${PATH}"

注意把安装路径从/Applications/Racket 7.4改为/Applications/Racket 也就是把数字去掉。

$ source ~/.zshrc

之后在命令行输入：

$ racket
Welcome to Racket v7.3.
>

这样，就算成功加入了环境变量。
