Linux统计文件夹和文件数目

2019-11-05
#Unix #Shell

统计当前目录下文件的个数（不包括目录）

ls -l | grep "^-" | wc -l

统计当前目录下文件的个数（包括子目录）

ls -lR | grep "^-" | wc -l

查看某目录下文件夹(目录)的个数（包括子目录）

ls -lR | grep "^d" | wc -l
