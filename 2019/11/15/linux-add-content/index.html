Linux下在文件内部指定行（首行、末尾行等）插入内容

2019-11-15
#Unix #Shell

1、在文件的首行插入指定内容：

sed -i "1i#! /bin/sh -" a

执行后，在a文件的第一行插入#! /bin/sh -

2、在文件的指定行（n）插入指定内容：

sed -i "niecho "haha"" a

执行后，在a文件的第n行插入echo “haha”

3、在文件的末尾行插入指定内容：

用第二种方法也可以。一般实现：

echo “haha” >> a

执行后，在a文件的末尾行插入haha
