2019-11-15
#Unix #Shell

删除：d命令

删除example文件的第二行：

sed '2d' example

删除example文件的第二行到末尾所有行：

sed '2,$d' example

删除example文件的最后一行：

sed '$d' example

删除example文件所有包含test的行：

sed '/test/'d example
