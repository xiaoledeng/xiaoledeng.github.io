LaTeX去掉图表的冒号

2019-04-23
#LaTeX

在LaTeX排版中，图表会有冒号，如，表 1.1: 或 图 1.1: 。

以下命令去掉此冒号：

\usepackage{caption}

\captionsetup[table]{labelsep=space} % 表
\captionsetup[figure]{labelsep=space} % 图

或者是：

\captionsetup{labelformat=default,labelsep=space} %去除冒号
