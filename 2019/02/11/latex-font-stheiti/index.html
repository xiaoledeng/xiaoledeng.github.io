LaTeX提示安装字体STHeiti

2019-02-11
#LaTeX

问题

在编译中文的文稿时，TeXpad编辑器一直提示：

x86_64-darwin needs to download the font “STHeiti”.

但是，点击下载后，发现这个弹窗一直存在，也就是没有下载成功。

系统环境

我的系统：Mac

LaTeX版本：MacTex 2018

LaTeX的编辑器：TeXpad

解决方案：

  1. 打开 FontBook的字体管理器，即为Font Book.app；
  2. 在FontBook里的“所有字体”中查找“STHeiti”；
  3. 然后下载STHeiti字体。
