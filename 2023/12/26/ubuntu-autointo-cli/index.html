Ubuntu默认进入命令行模式

2023-12-26
#Unix #Ubuntu

1. 引言

如果保持 Ubuntu 长期运行，默认进入命令行模式（command-line interface，CLI），有助于避免各种弹窗更新通知或桌面崩溃，而且能够节省开启桌面的资源。

2. 步骤

1.查看当前默认进入界面，终端输入：

systemctl get-default

显示：

graphical.target

即：图形界面。

2.关闭用户图形界面，使用字符模式登录：

systemctl set-default multi-user.target

显示：

==== AUTHENTICATING FOR org.freedesktop.systemd1.manage-unit-files ===
Authentication is required to manage system service or unit files.
Authenticating as: name,,, (name)

==== AUTHENTICATING FOR org.freedesktop.systemd1.reload-daemon ===

3.再次查看当前默认进入界面，终端输入：

systemctl get-default

显示：

multi-user.target

即：已经完成设置。

3.快速进入图形模式的命令

一种方法：

#当前立即进入字符模式
systemctl isolate multi-user.target

#当前立即进入图形模式
systemctl isolate graphical.target

另一种方法：startx，进入图形模式后，点击右上角 Log Out，可退出 X11 图形界面。

4. 关于图形模式

====================================
# 显示管理器启动、停止、查看状态
sudo service gdm3 start
sudo service gdm3 stop
sudo service gdm3 status
====================================
较新的gdm3使用最小版本的gnome-shell，并提供与GNOME3会话相同的外观。而gdm是GNOME显示管理器。

# 安装gdm3
sudo apt install gdm3
# 卸载gdm3
sudo apt remove gdm3

====================================
LightDM是Canonical的轻量级的、可配置的、显示管理器解决方案，它提供各种主题。
# 安装lightdm
sudo apt install lightdm
# 卸载lightdm
sudo apt remove lightdm

====================================
# 多显示模式切换
sudo dpkg-reconfigure gdm3
# 或
sudo dpkg-reconfigure lightdm
====================================

ubuntu安装gnome桌面
# 可在命令后面添加 --no-install-recommends 可选参数以避免非必要组件的安装，节省安装时间
sudo apt install ubuntu-desktop gnome-panel gnome-settings-daemon metacity nautilus gnome-terminal

====================================
