Yagmail设置Outlook邮箱

2023-12-18
#Python

根据 Outlook 官网 1 提示：

SMTP server name: smtp-mail.outlook.com
SMTP port: 587
SMTP encryption: STARTTLS

Yagmail 设置 Outlook 邮箱为：

yag_smtp_connection = yagmail.SMTP(
    user='name@outlook.com',
    host='smtp-mail.outlook.com',
    port=587, 
    smtp_starttls=True, 
    smtp_ssl=False
)
