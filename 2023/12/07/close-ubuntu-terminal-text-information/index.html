关闭Ubuntu终端打开时显示的文本信息

2023-12-07
#Unix #Shell #Fortran #Ubuntu

1. Ubuntu系统显示的信息

这个是Ubuntu系统显示的信息 1。禁用Ubuntu系统提示的信息：

touch ~/.hushlogin

2. oneAPI 启动显示的信息

这个是Ubuntu安装Intel® oneAPI Base & HPC Toolkit遗留的问题 2,3。

  1. 进入安装目录 /opt/intel/oneapi/，以管理员权限编辑setvars.sh：
cd /opt/intel/oneapi/
sudo vi setvars.sh
  1. 分别注释掉以下行输出到屏幕的代码：
# echo " "
# echo ":: initializing oneAPI environment ..."
# echo "   $sourcer"
# echo "   $input_args"

# echo ":: $arg_base -- $arg_verz"

# echo ":: oneAPI environment initialized ::"
# echo " "
  1. 修改 .bashrc 里的环境变量配置：
source /opt/intel/oneapi/setvars.sh intel64
