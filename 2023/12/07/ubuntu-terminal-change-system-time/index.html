Ubuntu终端修改系统时间

2023-12-07
#Unix #Shell #Ubuntu
  1. 查看当前系统时间
date -R
  1. 运行 tzselect
tzselect

依次选择：4) Asia；10) China；1) Beijing Time；1）Yes

  1. 复制文件到/etc目录下
sudo cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime
  1. 修改日期或时间
# 修改日期
sudo date -s MM/DD/YY
# 修改时间
sudo date -s hh:mm:ss
  1. 更新计算机硬件时间（将系统时间写入硬件时间）
sudo hwclock -w
  1. 查看时间：
timedatectl
