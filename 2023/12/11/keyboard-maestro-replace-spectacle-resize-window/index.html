Keyboard Maestro代替Spectacle调整窗口大小

2023-12-11
#Unix #Minimalism #Automation #macOS

1. 前言

一直以来，macOS 电脑💻的窗口大小和位置是通过 Spectacle 1 设置键盘⌨️快捷键快速调整，不需要鼠标🖱️，操作很方便。

开始使用 Keyboard Maestro 2 后，发现设置快捷键可实现同样的功能，故而 Spectacle 可以卸载了。

2. 设置步骤

  1. 新建一个 Macros，命名为 窗口左放

  2. 点击「New Trigger - Hot Key Trigger」，输入一个快捷键，例如： Option + Command + ->

  3. 点击「New Action」，进入 Categories 的「Interface Control」，选中「Resize a Window」，鼠标🖱️双击添加

  4. 点击「Resize by - Move and Resize」，选中「Left Column」

  5. 关闭「Actions」，完成✅操作

重复上面五步 3，完成 Keyboard Maestro 提供的九种最常见的窗口排列模式：

  • 全屏 (Full Screen)

  • 左右排列 (Left Column; Right Column)

  • 上下排列 (Top Half; Bottom Half)

  • 四角排列 (Top Left; Top Right; Bottom Left; Bottom Right)
