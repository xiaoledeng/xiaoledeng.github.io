macOS系统关闭Spotlight

2023-12-08
#Unix #macOS

注意⚠️：Spotlight关闭后，macOS系统的全文搜索🔍会失效。

1. 操作

  1. 关闭
sudo mdutil -a -i off
  1. 关闭快捷键

System Settings — Keyboard — Keyboard Shortcut — Sportlight — 关闭勾选 — Done

  1. 关闭Spotlight

System Settings — Siri & Spotlight — 关闭勾选所有选项

最后，补充打开操作：

sudo mdutil -a -i on
