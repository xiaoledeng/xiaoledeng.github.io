解决Ubuntu系统yagmail每次输入邮箱密码

2023-12-06
#Python #Unix #Ubuntu

之前写过用 Python 发送邮件的工具🔧 — yagmail 1。

但是，在 Ubuntu 系统上配置 yagmail 时，老是提示🔔 需要输入密码 2, 3。

解决方案如下：

yag_smtp_connection = yagmail.SMTP(
    user='user@qq.com',     # 发件人邮箱
    password='password',    # 邮箱密码或授权码
    host='smtp.qq.com'      # 邮箱的smtp地址
)

即，在 Python 文件中，直接加入和使用 password='password' # 邮箱密码或授权码。
