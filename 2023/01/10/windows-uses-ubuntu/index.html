Windows系统使用Ubuntu子系统

2023-01-10
#Unix #Windows #Ubuntu

1. 打开开发模式

Start > Settings > update & security > For developers

2. 打开子系统

Win + R: optionalfeatures.exe

勾选 Windows Subsystem for Linux

3. 下载 Ubuntu 系统

进入 Store，输入 Ubuntu，选择合适的版本下载。

4. 路径地址

Ubuntu 系统在 Windows 系统的路径为：

C:\Users\name\AppData\Local\Packages\CanonicalGroupLimited.Ubuntu22.04LTS_79rhkp1fndgsc\LocalState\rootfs
