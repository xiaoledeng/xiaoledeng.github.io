用不同的内核运行Mathematica程序

2023-01-26
#Mathematica #macOS #Unix #Windows #Ubuntu

1. 说明

在Windows系统和macOS系统里，默认用Local内核运行Mathematica程序。

对于多内核来说，可以定义不用的内核，即：用不同的内核运行不同的Mathmatica程序。

2. 步骤

  1. 添加新内核：Evaluation (计算) -> Kernel Configuration Options (内核配置选项)，有几个内核就设定几个

  2. 对于不同Mathamtica程序，选择不同的内核：Evaluation (计算) -> Notebook’s Kernel (笔记本内核)
