macOS系统远程Windows桌面

2023-01-23
#Unix #Windows #macOS

1. 步骤

第一步，先设置Windows系统能够被远程桌面访问。

  • 控制面板 -> 系统和安全 -> 允许远程访问
  • 高级设置里同样打开允许远程访问
  • 控制面板 -> 系统 -> 远程桌面

第二步，在App Center下载 Microsoft Remote Desktop Beta (macOS by Microsoft Remote Desktop)，并安装。

第三步，打开Microsoft Remote Desktop Beta软件，添加电脑，ip地址、用户名和密码。
