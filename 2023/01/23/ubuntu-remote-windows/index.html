Ubuntu系统远程Windows桌面

2023-01-23
#Unix #Windows #Ubuntu

1. 安装Ubuntu软件

sudo add-apt-repository ppa:x2go/stable

sudo apt-get update

sudo apt-get install x2goserver x2goserver-xsession

sudo apt install remmina

主要是：Remmina Remote Desktop Client软件。

2. Ubuntu操作命令

xfreerdp /u:username /p:password /w:1920 /h:1080 /v:ipaddress /smart-sizing +auto-reconnect

说明：

  • username 是用户名
  • password 是密码
  • ipaddress 是ip地址

注意：

  • Windows系统中要打开允许远程，见： How to enable Remote Desktop in Windows 10 (or Windows 7) | Digital Citizen
