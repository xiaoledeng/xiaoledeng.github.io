macOS系统利用samba访问Windows系统的共享文件

2023-01-23
#Unix #Windows #macOS

1. 步骤

前提：Windows系统的文件夹设置为共享模式。

第一步，Finder －> Go（前往） -> Connete to Server...（连接服务器）(Command + K)

第二步，输入IP地址、用户名和密码，连接即可。
