macOS系统调整图片

2023-05-06
#Unix #Experiences #macOS

1. 引言

图片调整，包括：

  • pixel, 像素
  • resolution, 分辨率, dpi
  • 文件大小
  • 文件格式（GIF, JPEG, JPG, PNG 等）

2. 调整工具

使用macOS系统自带的工具——Preview【预览】：

  • 打开照片
  • 工具栏 (Tools) -> 调整大小 (Adjust Size)
    • Fit into: pixels
    • Resolution: pixels
    • Resulting Size
  • 保存照片
