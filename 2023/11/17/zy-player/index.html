开源ZY-Player播放器

2023-11-17
#Unix #Experiences #Windows

1. 引言

ZY-Player 是电视直播和视频播放器 （ https://zyplayer.fun/ ），平台包括：

  • 🖥️ 电脑端 ( Windows & Mac & Linux )： ZY Player
  • 📱 手机端 ( Android & IOS )：ZY Player APP
  • 📺 电视端 ( Android & Mac ) ( 进行中 )：ZY Player TV
  • 🌐 浏览器 ( Web )：ZY Player Web

简而言之，ZY-Player是一个外壳工具🔧，通过视频源地址来观看视频。

关键是：视频源地址.json文件。一般，可以在网络搜索🔍。

2. 电脑端的ZY Player安装和操作

  1. 在 GitHub 下载 Release 2.8.8 Hunlongyu/ZY-Player 并安装软件

  2. 依次进入 ⚙️设置 — 源管理 — 编辑源，点击导入，选中本地json文件，点击检测

  3. 点击最左边的圆圈，进入主页面，选中资源（例如，樱花资源），在站内/组内/全站进行搜索（例如，站内搜索 哈利波特），进入页面，点击播放
