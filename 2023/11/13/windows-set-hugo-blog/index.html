Windows系统搭建Hugo博客

2023-11-13
#Blog #Windows

1. 步骤

  1. 先安装 Hugo，下载 hugo.exe 可执行文件后，关键是添加 可执行文件 到 Windows系统的PATH路径；

  2. 测试安装 Hugo 是否成功，终端输入：

hugo version
  1. 终端输入：
hugo new site blog

  1. 下载 themes 文件夹：xiaoledeng/simple-style，放到 themes 文件夹。

  2. 回到根目录，终端输入：

hugo server -t simple-style --buildDrafts

  1. 根据提示，打开网址： http://localhost:1313/ ，就可以看到网站基本内容。

  2. 博客文件在 content/post/文件夹，格式为 markdown格式，头文件如下：

---
title: "标题"
slug: title
date: 2012-01-01T01:00:00+08:00
tags:
    - "Life"
    - "Ideas"
---
