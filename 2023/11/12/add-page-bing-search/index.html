个人网站加入Bing搜索

2023-11-12
#Blog

步骤

  1. 先注册账户： https://www.bing.com/webmasters

  2. 使用 XML 文件验证，先将 XML 文件下载，再将该文件上载到网站的根目录中

  3. 在网页上验证

  4. 验证成功后，提交网站地图： https://www.xiaoledeng.com/sitemap.xml
