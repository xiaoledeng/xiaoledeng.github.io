Ubuntu系统桌面重启

2023-02-24
#Unix #Ubuntu

1. 说明

可能是磁盘空间容量不足，导致Ubuntu系统的桌面卡住，需要在不重启电脑的前提下，重启桌面。

注意⚠️：不要用电脑按钮强制关机，因为很有可能再次开机后系统文件损坏，无法进入系统。

2. 步骤

第一步，切换到tty终端，Ctrl + Alt + F1、Ctrl + Alt + F2、Ctrl + Alt + F3等，进入一个即可。

第二步，登陆到用户。

第三步，输入一下命令：

sudo pkill Xorg

或

sudo restart lightdm

等待⌛️几分钟，即可出现新的桌面登陆界面。
