Ubuntu系统磁盘工具 Disk Usage Analyzer

2023-02-24
#Unix #Ubuntu

Ubuntu系统的日志文件居然能够达到120G左右，导致剩余磁盘空间没多少。

可以定期用磁盘工具Disk Usage Analyzer查看异常文件，定位具体路径，而后用命令行删掉：

sudo rm /var/log/syslog.1

或

sudo rm -rf /var/log/path/
