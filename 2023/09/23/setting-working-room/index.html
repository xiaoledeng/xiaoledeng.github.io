随笔—布置办公空间

2023-09-23
#Experiences #Essay

1. 前言

在专注也需要一个空间：给自己布置一个简易书房角落 - 少数派的基础上，布置了办公空间。

营造一个专注的空间，舒适高效，是非常有必要的。

2. 关键物品

  • 桌：优先是乐歌升降桌，尺寸至少是 120cm x 60cm。另外配件包括：隐藏抽屉盒。
  • 椅🪑：人体工学椅，比如，网易严选入门款。另外配件包括：垫子 + 背靠。
  • 灯💡：护眼台灯 + 屏幕挂灯
  • 显示器🖥️：优派 27in 4k VX2780-4k-HD-2
  • 摇臂支架：NBF160-FD 可以放一台显示器和笔记本电脑。如果显示器多，可以选四个，甚至八个。
  • 电脑💻配件：机械键盘⌨️ + 鼠标🖱️ + 拓展坞 + 桌面延长板
  • 洞洞板及其配件：主要是利用立式空间收纳
  • 置物架：大的置物架，配上若干收纳盒，可以放 打印机🖨️ + 收纳盒 + A4文件架等。
  • 插排收纳盒：公牛插排收纳盒，隐藏若干充电数据线，干净整洁，安全。
  • 书架：传统书架
  • 闹钟⏰：显示时间
  • 香薰：栀子花味的香薰
  • 收音机📻音箱：猫王蓝牙收音机音箱，有颜值，音效好
