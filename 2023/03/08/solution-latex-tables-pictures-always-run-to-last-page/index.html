LaTeX表格或图片太多总是跑到最后一页的解决方案

2023-03-08
#LaTeX

1. 说明

在LaTeX文档中，表格或图片太多，总是跑到参考文献的后面，即跑到最后一页。

2. 解决方案

在文档最前面添加以下包：

\usepackage[section]{placeins}

作用：禁止图片或者表格浮动超过\section的范围
