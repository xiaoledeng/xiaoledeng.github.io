macOS系统命令行运行Mathematica

2023-03-16
#Unix #Mathematica #macOS

1. 核心步骤

将 Mathematica 的安装目录添加到 PATH 环境变量中，打开终端应用程序，输入以下命令：

vi ~/.zshrc

在打开的文件中添加以下行：

export PATH=$PATH:/Applications/Mathematica.app/Contents/macOS

激活环境变量的路径：

source ~/.zshrc

2. 注意⚠️

在以前的博文服务器上运行Mathematica命令行代码，终端中打开Mathmatica的命令行为 math。

但是，现在变为：wolfram、WolframKernel和MathKernel，还有：wolframscript。详细的命令可以在路径 /Applications/Mathematica.app/Contents/macOS查找。

macOS系统下Mathematica版本如下。

wolfram、WolframKernel和MathKernel启动信息为：

Mathematica 12.3.1 Kernel for Mac OS X ARM (64-bit)
Copyright 1988-2021 Wolfram Research, Inc.

wolframscript启动信息为：

Wolfram Language 12.3.1 Engine for Mac OS X ARM (64-bit)
Copyright 1988-2021 Wolfram Research, Inc.

Mathematica 和 MathematicaServer 则是打开Mathematica的记事本。
