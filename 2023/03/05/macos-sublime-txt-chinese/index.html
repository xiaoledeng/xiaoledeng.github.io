macOS系统Sublime打开txt文件的中文乱码的解决方案

2023-03-05
#Unix #macOS

1. Sublime设置

  1. Control + Shift + P 进入 Package Control: Install Package；

  2. 输入 Install Package，输入 ConvertToUTF8 插件，重启即可。

注意⚠️：对于Sublime 3，还需要安装 Codecs33插件。

Encoding: GB2312，Error: Codecs missing，Please install Codecs33 plugin (https://github.com/seanliang/Codecs33/tree/osx) or upgrade to Sublime Text 4.

2. macOS系统默认文本TextEdit编辑器的设置

Settings -> Open and Save -> Plain Text File Encoding -> Opening Files，改为 Chinese (GB 18030)，即可。
