macOS安装Python科学计算工具包Anaconda

2017-01-04
#Python #Unix #macOS #Software

入门知识： Python 科学计算发行版 –Anaconda | SeisMan

1. 下载安装

在官网上面下载 Download Anaconda Now! | Continuum。

2. 申请免费的学术 License

申请地址： https://store.continuum.io/cshop/academicanaconda

在User Settings > Add-ons下载三个license：

  • license_accelerate_1483528291.txt
  • license_mkl_1483528125.txt
  • license_iopro_1483528294.txt

把上面的三个文件拷贝到~/.continuum目录下：

  cp -i /Users/name/Downloads/*.txt   ~/.continuum

而后就OK啦。

3. pip

pip 是 Python 自带的模块安装工具，比如:

pip install requests
pip install requests --upgrade
