终极参考文献引用管理—BibTeX

2017-01-10
#LaTeX

就我而言，关于文献管理，在科研写作中，就是在Reference部分中排版，然后在论文中适当地格式引用。

最早用文献管理的软件是NoteExpress，而后是Endnote，现在是BibTex。

关于Endnote，批量参考文献的引用撰写了一个文稿：批量参考文献格式的引用技巧。Endnote适合在Windows下面的Word里面来引用，对应不同杂志的格式，在Endnote里面来变换不同的格式，将增加大量的重复工作。

现在，在Mac下面使用MacTex套装来科研写作。对于参考文献的管理和引用，得跟得上国际步伐，故此使用Latex工作流，发现并采用BibTeX格式的参考文献管理。

可以这样坦诚以及很负责任地说：一旦掌握了基本的BibTeX用法后，剩下的未来科研写作人生，都不需要自己去调整参考文献的格式了，只需要找到对应的模板，而后更换一行命令为对应模板即可。

1. 下载bib格式的参考文献格式

  • 在每个杂志的电子文档页面都会有导出参考格式；
  • 当然GoogleScholar也有导出BibTeX的格式，方法类似于：批量参考文献格式的引用技巧。
  • LaTeX技巧857：如何找到标准的参考文献Bib文件-LaTeX工作室

将所有的BibTeX内容都复制到一个格式为bib的文件中，如mybibex.bib 。这个mybibex.bib就是包括了你写文章所有的参考文献，也可以说是你一生科研写作的参考文献库。

2. 编译引用技巧

  • 参考文献的内容放在：mybibex.bib文件；
  • natbib.bst格式的文件，即为参考文献样式文件，针对不同的杂志会有不同的样式，需要自己Google到对应的bst文件。
  • 主文件a.tex文件。

在主文件a.tex中，参考文献部分插入以下代码：

% BibTeX References
\bibliographystyle{natbib}  
\bibliography{mybibtex}   % name my BibTeX data base

注意：

  • \bibliographystyle{natbib} 为参考文献的样式，注意为natbib，不需要后缀；
  • \bibliography{mybibtex} 为个人的参考文献库。
  • 在a.tex其他的位置，需要有\cite{文献a}、\citet{文献a}、\citep{文献a} 等的引用格式，否则参考文献不会显示；
  • 编译a.tex的时候，LaTex—BibTeX—LaTex—LaTex，需要编译四次。

详细地论述，可以参考LaTeX技巧829:使用BibTeX生成参考文献列表-LaTeX工作室。
