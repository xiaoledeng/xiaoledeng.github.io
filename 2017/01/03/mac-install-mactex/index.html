Mac安装MacTex

2017-01-03
#LaTeX #Unix

1. 下载MacTex并安装

下载官网的MacTex，并安装。

2. Latex编译器TexShop的设置

安装完成后，会有TexShop编译器。

完成如下两个中文的环境设置：

  • 设置默认的编译为Xelatex：Preferences—Typesetting—Default Command—Command Listed Below—XeLaTex；

  • 设置语言编码为UTF-8：Preferences—Source—Encoding—Unicode(UTF-8)；当然Text and Background Colors也可以设置为白字黑底；

3. 测试中英文混搭

将下面的代码保持为UTF-8的Hello.tex文档。

  \documentclass{article}
  \usepackage{fontspec}
  \setmainfont{SimSun}

  \begin{document}

  Hello, \LaTeX3, 中文混搭效果。

  \end{document}

编译快捷键command+T，编译生成PDF格式。
