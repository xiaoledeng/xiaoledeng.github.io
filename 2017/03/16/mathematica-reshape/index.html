Mathematica的Reshape矩阵重塑

2017-03-16
#Mathematica

Reshape命令

一般需要对矩阵进行重新分配行列，如转置等。需要如下命令：

ArrayReshape[list,dims]

解释：将 list 中的元素排列成一个维数为 dims 的矩形数组。

其中list可以表示为矩阵形式；dims为{x,y}维数。

 In[1]:=     ArrayReshape[{a, b, c, d, e, f}, {2, 3}]
 Out[1]:=    {{a, b, c}, {d, e, f}}
