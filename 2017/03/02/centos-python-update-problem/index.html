服务器Centos(Linux)上Python2-6-6升级到2-7-6问题

2017-03-02
#Unix

服务器Centos(linux)上python2.6.6升级到2.7.6，yum不能用了的解决办法：

  vim /usr/bin/yum

编辑第一行

  #!/usr/bin/python
  import sys
  try:

改成如下（原python版本是2.6）

  #!/usr/bin/python2.6
  import sys
  try:

这些yum正常了。
