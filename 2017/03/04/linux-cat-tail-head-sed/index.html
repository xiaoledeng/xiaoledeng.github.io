cat、tail、head、sed查看文件任意几行的数据

2017-03-04
#Unix #Shell

一、使用cat、tail、head组合

1、查看最后1000行的数据

cat filename | tail -n 1000

2、查看1000到3000行的数据

cat filename | head -n 3000 | tail -n +1000
  • cat filename 打印文件所有内容
  • tail -n 1000 filename 打印文件最后1000行的数据
  • tail -n +1000 filename 打印文件第1000行开始以后的内容
  • head -n 1000 filename 打印前1000的内容

二、使用sed命令

显示1000到300行的数据

sed -n '1000,3000p' filename
