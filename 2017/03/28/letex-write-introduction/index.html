LaTeX科研写作入门手册

2017-03-28
#LaTeX

前言

使用LaTeX进行学术写作，够用就行。也就是说，LaTeX只是一个排版的工具，科研学术写作重要的是内容。

本文作为从“不知道LaTeX的小白”到“能够使用LaTeX进行写作”的简明入门手册，也是自己摸索出来的高效率学习LaTeX的方法。

首先，请阅读文章，明白个人在使用LaTeX学术写作的一些理解和思考：

  • 科研高效率(12)—LaTeX式科研方式
  • 学习Latex的一点想法

切记：不要自己从零开始学习LaTeX，完全没必要。

入门步骤

  • Linux系统——TeXLive2017；推荐下载地址：TeXLive Download，下载当前目录中的压缩文件：

LaTeX编辑器推荐

优秀的LaTeX编辑器能够大大提高科研写作过程中的效率，特别是在tex编译成pdf之后的对照查看阶段。

Mac下面特别推荐的LaTeX顺序： TeXstudio > Texpad > Sublime Text + Skim 组合 > TeXshop

LaTeX的模板

下载所需要的模板文件（注：一般是压缩包，解压，得到很多文件），打开解压文件夹中的tex主文件，编译并调试，直到运行成功（即自动弹出PDF文件）。

一般而言，官网上会有 LaTeX 模版。

Tex文件和PDF文件的结构

对照PDF中模板文章的结构，观察tex代码的的结构，而后改写成自己的文章。

下面以AGU通用模板为例子，来说明Tex文件和PDF文件的结构。

先下载 agu-template-updated-20161103.zip ，并解压如下图：

这里，\*.tex都属于源文件，即为要打开编辑、编译的主文件；agufull08.bst和agujournal.cls都属于格式文件，一般不需要修改；figsamp.eps和figsamp.pdf是图片。

投稿文章的结构包括：

  1. 标题、作者、单位等文章开头信息；
  2. 摘要和关键词（Abstract、Keywords）；
  3. 文章主要部分（Introduction、Theory、Application、Discussions and Conclusions）；
  4. 致谢（Acknowledgments）；
  5. 参考文献（References）；
  6. 附录（Appendix）；

  • 开头信息

  • LaTeX 三剑客——图、表和公式。图和表会有通用的插图和插表的模块；公式可以在线编辑（推荐：LaTeX公式在线编辑器）,然后复制代码即可。

  • 参考文献（References），可以参考 终极参考文献引用管理—BibTeX 。
