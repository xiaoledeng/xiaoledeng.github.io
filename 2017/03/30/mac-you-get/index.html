Mac下音频下载工具You-Get

2017-03-30
#Unix #Experiences

本文对于Mac下，音频下载工具You-Get的介绍。

1. 安装

请确保已经安装了python3和FFmpeg。之后，命令行输入：

pip3 install you-get

或者输入：

brew install you-get

之后就安装好了。

2. 视频列表

据You-Get官网介绍，You-Get支持下载包括YouTube, Youku, Niconico在内的70余家视频网站，其中还有有丰富技术视频教程的Khan Academy、InfoQ。具体见列表：supported-sites。

3. 使用

  1. 查看所有可以下载的格式：

    $ you-get -i ‘https://www.youtube.com/watch?v=jNQXAC9IVRw'

  2. 下载mp4格式（–itag=18）：

    $ you-get –itag=18 ‘https://www.youtube.com/watch?v=jNQXAC9IVRw'
