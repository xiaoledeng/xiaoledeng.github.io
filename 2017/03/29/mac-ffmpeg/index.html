Mac下音频格式转换工具FFmpeg

2017-03-29
#Unix #Experiences

Mac命令行的FFmpeg命令可以用来转换视频格式。

1. 安装FFmpeg

通过以下命令来安装：

brew install ffmpeg

2. 常用命令

  • 获取媒体信息命令：
ffmpeg -i video.avi
  • 从视频文件中提取声音信息，并存为Mp3格式
ffmpeg -i source_video.avi -vn -ar 44100 -ac 2 -ab 192k -f mp3 sound.mp3

参数解释 :

Source video : source_video.avi

Audio bitrate : 192kb/s

output format : mp3

Generated sound : sound.mp3
