LaTeX设置字体大小

2017-03-29
#LaTeX

全局模式

    \documentclass[12pt]{article}

在文档的开头，有设置整个文章的字体大小，如：12pt。

局部模式

设置字体大小的命令从小到大为：

    \tiny
    \scriptsize
    \footnotesize
    \small
    \normalsize
    \large
    \Large
    \LARGE
    \huge
    \Huge

注意：局部模式是相比于全局字体的基础上，来变大变小。

例子：

    \documentclass[12pt]{article}
    \begin{document}

    Hello Latex.

    \tiny Hello Latex.
    
    \scriptsize Hello Latex.
    
    \footnotesize Hello Latex.
    
    \small Hello Latex.
    
    \normalsize Hello Latex.
    
    \large Hello Latex.
    
    \Large Hello Latex.
    
    \LARGE Hello Latex.
    
    \huge Hello Latex.
    
    \HUGE Hello Latex.

    \end{document}
