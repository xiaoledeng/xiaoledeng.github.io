Python的统计函数——最大值、最小值、平均值和标准差

2017-04-13
#Python

Python下，可以直接对一个列表a进行统计分析而得到此列表a的最大值、最小值、平均值和标准差：

a.max()      #最大值
a.min()      #最小值
a.men()      #平均值
a.std()      #标准差
