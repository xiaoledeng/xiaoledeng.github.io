LaTeX参考文献管理工具BibDesk

2017-04-11
#LaTeX

之前在 终极参考文献引用管理——BibTeX 中论述到了：使用BibTex格式的文件来统一管理Latex的参考文献。

那么，对于Bib文件，Mac下MacTex套装中有一个工具——BibDesk，来管理这样一个Bib文件。

TeXShop制作参考文献_百度经验初步介绍了一点点用法。

这里，要介绍一下内置的External-Web(BibDesk Web Group)浏览器，来自动导入Bib文件。这样就告别了下载-导入的限制，实现了半自动化的参考文献管理。

具体操作见下图：

1、先到浏览器输入网址，即为bib的下载网址；

2、点击下方的Import按钮即可导入。
