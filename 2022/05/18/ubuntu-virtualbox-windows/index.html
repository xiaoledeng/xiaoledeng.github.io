Ubuntu下用VirtualBox虚拟机安装Windows系统

2022-05-18
#Unix #Windows #Ubuntu

1. 下载并安装VirtualBox虚拟机

第一种方法，在官网 https://www.virtualbox.org/wiki/Linux_Downloads 下载并安装VirtualBox虚拟机。

第二种方法，用命令行：

sudo apt-get install virtualbox

2. 安装Windows系统并设置

设置语言：打开virtualbox，File—Perferences—Language—简体中文。

新建虚拟机并设置：点击新建，并设置虚拟机名称和存放位置、CPU内存、磁盘空间。

挂载ISO文件：点击设置—存储—没有盘片—选择虚拟磁盘。

配置windows虚拟机屏幕大小：点击设备—安装增强功能。
