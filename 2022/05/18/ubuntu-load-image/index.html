Ubuntu打开图片Could not load image...0x89 0x50错误

2022-05-18
#Unix #Ubuntu

1. 错误详情

Ubuntu打开手机截图的图片，发现如下错误：

Error interpreting JPEG image file (Not a JPEG file: starts with 0x89 0x50)

2. 解决方案

图片文件后缀改为png即可。
