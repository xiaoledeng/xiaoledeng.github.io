macOS个性化设置Time Machine的备份策略

2022-02-21
#Unix #macOS

1. 前言

备份的重要性，怎么强调也不过分。对于科研工作者来说，定期备份数据很重要，尤其是在每日工作机器上。

macOS系统上的Time Machine工具，默认是：

自动制作过去 24 小时的每小时备份、过去一个月的每天备份以及之前所有月份的每周备份。

想根据自己的要求来定制备份频率和策略等，需要额外的工具——TimeMachineEditor。

2. 安装和设置

brew install --cask timemachineeditor

TimeMachineEditor有三种模式：

  1. When inactive——机器处于闲置时会自己进行备份
  2. Interval——时间间隔，以小时、天、周为单位
  3. Calender Intervals——设置哪天什么时候备份

注意：

  • 需设置电池或电源的Power Nap保持常开
  • 关掉Time Machine的 Back Up Automatically
