3. 基本设置

更换软件源：

sudo pacman-mirrors -i -c China -m rank

更新软件库：

sudo pacman -Syy

更新系统：

sudo pacman -Syu

注意：不建议配置archlinuxcn的源。

必须要做timeshift （备份回滚系统的工具），防止重装：

sudo pacman -S timeshift

其他软件：

sudo pacman -S bat
sudo pacman -S neofetch
sudo pacman -S htop
sudo pacman -S btop
sudo pacman -S duf
