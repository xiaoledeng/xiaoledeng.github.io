Ubuntu利用aptitude自动解决依赖包问题

2022-02-25
#Unix #Ubuntu

1. 说明

当Ubuntu用sudo apt install ...安装软件时，会遇到依赖包的问题。

2. 解决

把apt换为aptitude，即：

sudo aptitude install ...

当然，还有一个办法，利用snap去管理软件。
