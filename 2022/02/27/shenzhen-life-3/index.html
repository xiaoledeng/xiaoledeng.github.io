记深圳居家办公的日子(3)—Ubuntu利用Timeshift进行系统备份与还原

2022-02-27
#Life #Unix #Ubuntu

1. 安装

sudo apt-add-repository -y ppa:teejee2008/ppa
sudo apt-get update
sudo apt-get install timeshift

2. 备份

打开Timeshift，进行基本的设置，包括：

  • 备份文件格式：【RSYNC】
  • 选取备份文件位置：【位置】，硬盘文件格式要为Linux文件系统ext4，Windows的NTFS格式不支持
  • 设置备份计划频率：【计划】
  • 【用户】和【筛选】选择默认即可

3. 还原

3.1 能进入系统

直接打开Timeshift，选择合适节点，进行还原。

3.2 能进入终端

1.Ctrl+Alt+F1进入终端

2.查看可以还原节点

sudo timeshift --list

3.选择节点（如：2019-07-16_16-35-42）进行还原

sudo timeshift --restore --snapshot '2019-07-16_16-35-42' --skip-grub

或

sudo timeshift --restore --snapshot '2019-07-16_16-35-42' --skip-grub --scripted

3.3 无法进入系统，通过U盘启动系统

1.制作Linux live CD的U盘系统启动盘

2.进入U盘的Live CD系统，打开Timeshift软件，点击设置按钮，设置快照的存储位置

3.按照3.1节内容设置还原即可
