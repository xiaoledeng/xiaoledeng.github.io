macOS清理系统数据

2022-02-22
#Unix #macOS

2. OmniDiskSweeper工具和du命令

brew install omnidisksweeper

查看当前文件夹下各个文件和文件夹占用的空间大小：

du -sh *

3. 说明

系统缓存及日志文件位置：

  • 系统缓存保存在：~/Library/Caches
  • 系统日志保存在：~/Library/Logs

应用缓存及日志文件位置：

  • 缓存文件保存在：~/Library/Containers/com.xx.xx(应用名称)/Data/Library/Caches
  • 日志文件保存在：~/Library/Containers/com.xx.xx(应用名称)/Data/Library/Logs

清除缓存文件:

cd ~/Library/Caches/
rm -rf ~/Library/Caches/*

删除临时文件:

cd /private/var/tmp/
rm -rf TM*

删除所有系统日志:

sudo rm -rf /private/var/log/*
