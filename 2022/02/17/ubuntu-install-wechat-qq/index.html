Ubuntu安装微信和QQ

2022-02-17
#Unix #Ubuntu

1. 优麒麟的ukylin-wine和ukylin-wechat

地址： http://archive.ubuntukylin.com/software/pool/partner/

下载：

wget http://archive.ubuntukylin.com/software/pool/partner/ukylin-wine_70.6.3.25_amd64.deb
wget http://archive.ubuntukylin.com/software/pool/partner/ukylin-wechat_3.0.0_amd64.deb
wget http://archive.ubuntukylin.com/software/pool/partner/ukylin-qq_1.0_amd64.deb

安装：

sudo apt-get install -f -y ./ukylin-wine_70.6.3.25_amd64.deb
sudo apt-get install -f -y ./ukylin-wechat_3.0.0_amd64.deb
sudo apt-get install -f -y ./ukylin-qq_1.0_amd64.deb

结论：微信可以正常使用；QQ的部分字体缺失 (中文乱码是空格)，但不影响使用。

解决方案：

sudo apt-get install fonts-droid-fallback ttf-wqy-zenhei ttf-wqy-microhei fonts-arphic-ukai fonts-arphic-uming

下载字体： YaHeiConsolas.tar

tar -zxvf YaHeiConsolas.tar.gz

sudo mkdir -p /usr/share/fonts/vista

sudo cp YaHeiConsolas.ttf /usr/share/fonts/vista/

sudo chmod 644 /usr/share/fonts/vista/*.ttf

cd /usr/share/fonts/vista/

sudo mkfontscale && sudo mkfontdir && sudo fc-cache -fv

sudo apt-get install unity-tweak-tool

利用 unity-tweak-tool 设置 微软雅黑 + Consolas 字体。

2. 网页版微信 electronic-wechat

sudo apt-get install snap
sudo snap install electronic-wechat

结论：网页版微信不能登陆。

3. 深度系统的wine

wget -O- https://deepin-wine.i-m.dev/setup.sh | sh  
# Tencent WeChat Work Client on Deepin Wine6
sudo apt-get install com.qq.weixin.work.deepin
# 微信 Tencent WeChat Client on Deepin Wine
sudo apt-get install com.qq.weixin.deepin
# Tencent TIM Client on Deepin Wine
sudo apt-get install com.qq.office.deepin
# QQ Tencent QQ Client on Deepin Wine6
sudo apt-get install com.qq.im.deepin

深度系统的软件包列表： https://deepin-wine.i-m.dev/

  • 首次使用需要添加仓库： wget -O- https://deepin-wine.i-m.dev/setup.sh | sh
  • 而后使用： apt install … 安装对应软件包

结论：软件包间的依赖关系无法满足，不能安装。
