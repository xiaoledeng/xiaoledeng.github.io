Install ibus on Ubuntu

2022-02-16
#Unix #Ubuntu

1. Install ibus

sudo apt-get install ibus ibus-clutter ibus-gtk ibus-gtk3 ibus-qt4
im-config -s ibus
sudo apt-get install ibus-pinyin

2. Set language support

Open Language Support, choose Keyboard input method system to IBUS, and Apply System-Wide.
