Install Julia on macOS

2022-02-02
#Julia #Unix #macOS

1. Download

At the official website https://julialang.org/downloads/, download the file julia-1.7.1-mac64.dmg.

Drag and install Julia in the traditional way of macOS.

2. Add Julia to PATH

sudo mkdir -p /usr/local/bin
sudo rm -f /usr/local/bin/julia
sudo ln -s /Applications/Julia-1.7.app/Contents/Resources/julia/bin/julia /usr/local/bin/julia

or open ~/.zshrc and add this command:

alias julia=\''/Applications/Julia-1.7.app/Contents/Resources/julia/bin/julia'\'
