Ubuntu查看系统安装时间、硬盘信息、设备序列号、MAC地址和IP地址

2022-07-04
#Unix #Ubuntu

1. 系统安装时间

ls /dev | grep sd
sudo dumpe2fs /dev/sda2 | grep "Filesystem created"

2. 硬盘信息

sudo hdparm -i /dev/sda1

其中，Model为型号，SerialNo为硬盘序列号

3. 设备序列号

sudo dmidecode -t 2 | grep Serial

4. MAC地址

sudo lshw -c network | grep serial | head -n 1

5. IP地址

ifconfig
