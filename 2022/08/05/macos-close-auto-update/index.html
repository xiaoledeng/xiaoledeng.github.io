macOS系统关闭Microsoft AutoUpdate更新

2022-08-05
#Windows #Unix #Experiences #macOS

1. 方法

方法一：设置权限不可访问

cd /Library/Application\ Support/Microsoft/MAU2.0
sudo chmod 000 Microsoft\ AutoUpdate.app

或

sudo chmod -R 000  /Library/Application\ Support/Microsoft/Microsoft\ AutoUpdate.app

方法二：删除Microsoft\ AutoUpdate.app

sudo rm -rf /Library/Application\ Support/Microsoft/Microsoft\ AutoUpdate.app

2. 打开更新

下载并安装 Microsoft AutoUpdate tool，在软件里面更新即可。
