macOS系统关闭Microsoft AutoUpdate更新

2022-08-05
#Windows #Unix #Experiences #macOS

1. 方法

方法一：设置权限不可访问

cd /Library/Application\ Support/Microsoft/MAU2.0
sudo chmod 000 Microsoft\ AutoUpdate.app

或

sudo chmod -R 000  /Library/Application\ Support/Microsoft/Microsoft\ AutoUpdate.app

方法二：删除 Microsoft\ AutoUpdate.app

sudo rm -rf /Library/Application\ Support/Microsoft/Microsoft\ AutoUpdate.app

方法三：在系统层面禁用启动项

# 禁用 AutoUpdate 的自动启动
launchctl unload -w /Library/LaunchAgents/com.microsoft.update.agent.plist 2>/dev/null
sudo launchctl unload -w /Library/LaunchDaemons/com.microsoft.autoupdate.helper.plist 2>/dev/null

# 手动删除它的后台常驻组件（不会影响 Office 使用）
sudo rm -rf "/Library/Application Support/Microsoft/MAU2.0"
sudo rm -f "/Library/LaunchAgents/com.microsoft.update.agent.plist"
sudo rm -f "/Library/LaunchDaemons/com.microsoft.autoupdate.helper.plist"

2. 打开更新

下载并安装 Microsoft AutoUpdate tool，在软件里面更新即可。
