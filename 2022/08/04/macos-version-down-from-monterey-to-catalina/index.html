macOS系统从Monterey 12.3降到Catalina 10.15.7

2022-08-04
#Unix #macOS

1. 引言

最近，明显感觉macOS系统变卡了。

原因是：2015 Macbook Pro 13寸安装的Monterey 12.3系统不是很流畅，硬件带不动软件。

在知乎搜索了一下，得到以下答案：

Monterey的确是给 m1 pro/max 准备的，建议 intel 芯片的就不要升级了。

用的最多的应该是 macOS Mojave 10.14.x、Catalina 10.15.x 了，非 M1 芯片的 intel 机器，选这两个就好了。

机器出厂版本的最后一个小版本比较好（不升大版本）。

若追求系统稳定建议升级到 macOS Catalina 10.15.7。

Catalina是macOS 10里的最后一个版本，Catalina能比Mojave能获得更久的APP适配支持。如果为了长远的软件适配考虑，想尽量选一个有更长生命周期的版本的话，Catalina是那个更适合Mac养老的版本。

故此，综合考虑流畅性、稳定性、兼容性和安全性（安全性更新 2022-005 Catalina，发布时间：2022 年 7 月 20 日），选择：Catalina 10.15.7系统。

2. 操作流程

第一，备份系统的数据。

第二，启动时使用 Shift-Option-Command-R 进入 macOS 恢复的实用工具窗口。

第三，进入硬盘工具，抹掉磁盘，注意是内部盘。

第四，返回 macOS 恢复的实用工具窗口，选择重新安装系统，这时应该是安装出厂版本的系统。我的是OS X El Capitan 10.11。

第五，安装完OS X El Capitan 10.11系统后，打开Safari浏览器地址如何获取旧版 macOS - Apple 支持 (中国)，点击macOS Catalina下载系统，而后安装操作完成安装。

安装完成后，Catalina 10.15.7系统用起来变流畅。

3. 注意事项

  • 抹掉磁盘前，一定要备份好数据；
  • 全程不需要额外的硬件，不过需要在网络环境下操作；
  • OS X El Capitan 10.11系统太老了，虽然很流畅，但是很多软件安装不了，除非软件安装包有备份；
  • 硬件决定了系统，系统决定了软件，流畅性和稳定性是电脑高效工作的前提；另外，兼容性和安全性也是必须要保证的。
