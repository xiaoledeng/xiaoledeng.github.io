macOS系统youtube-dl下载视频、音频和字幕

2022-08-11
#Unix #Experiences #Automation #macOS

1. 安装youtube-dl

brew install youtube-dl

2. 下载视频

youtube-dl -f best URL

其中，URL为视频地址。

3. 下载音频

最佳音频下载：

youtube-dl -f bestaudio URL

4. 下载字幕

列出字幕：

youtube-dl --list-subs URL

下载字幕：

youtube-dl --write-sub --sub-lang de-DE --skip-download URL

de-DE可以改为第一步列出的字幕名称，比如：zh-Hans、zh-Hant和en等。
