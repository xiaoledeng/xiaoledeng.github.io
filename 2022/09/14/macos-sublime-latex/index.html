macOS系统Sublime配置LaTeX编译环境

2022-09-14
#Unix #LaTeX #macOS

1. 安装LaTeXTools

按键 command + shift + p，调出 Package Control install package，再输入：LaTeXTools，完成安装。

2. 安装Skim PDF阅读器

在Skim | Home下载dmg，完成安装。

3. 配置Skim

打开Skim——Perferences——Sync——PDF-TeX Sync support 的 Preset 设置为 Sumblime Text。

同时，勾选 Check for file changes和Reload automatically。
