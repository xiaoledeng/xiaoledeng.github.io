成果出图的技能(2)—脚本画图

2016-11-14
#Essay #Experiences

在地球科学领域的作图方面，不得不提到一个软件：GMT软件。官方网址：Wiki - GMT - GMT — The Generic Mapping Tools

此软件是通过脚本来进行图片细节方面的控制，非常便于高效地绘图。

同一个类型的图，只要编辑好画图脚本，之后在绘制此类的图片时，直接在之前的脚本上稍作修改就行。换句话说，只要一次绘制成功一个图形，那么之后此类图型都不是问题。

官网上面有很多的教程，当然是英文的：Documentation - GMT - GMT — The Generic Mapping Tools

另外，中文也有翻译的官网教程，推荐一个博客：SeisMan。
