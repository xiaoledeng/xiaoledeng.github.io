Fortran调试(3)—表达式太长

2016-11-23
#Fortran

调试Fortran代码的时候，遇到了一个问题：

  catastrophic error: Statement too long

就是一个表达式太长，Google之后，发现了这个：intel Fortran catastrophic error: Token too long, max is 7194 - Stack Overflow。

所以解决办法：

  • 切割表达式为几段。
  • 输入的表达式化简，化简，化简(Simply or FullSimply)！！！
