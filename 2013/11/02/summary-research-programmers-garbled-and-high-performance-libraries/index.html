2. 如何成为一名优秀的程序员

PS: 找东西的时候发现的：

Q:如何成为一名优秀的程序员？

A（Cleve Moler）：要成为一名优秀的程序员，首先要去看别人写的优秀的程序，你也可以去看我写的程序。就像当你要学习写作时，你首先是去读别人的作品，尤其是那些大作家的作品，莎士比亚，海明威等，而不是直接就坐下来直接写自己的作品。当然也要看一下那些不好的程序，要明白他们为什么不好。

3. Ubuntu下打开txt文件乱码的解决方法

1、终端输入 sudo apt-get install dconf-tools 安装，然后输入sudo dconf-editor调出dconf-editor

2、依次打开org->gnome->gedit->preferences->encodings中的auto-detected的value改为:['GB18030', 'GB2312', 'GBK', 'UTF-8', 'CURRENT', 'ISO-8859-15', 'UTF-16'],直接回车保存就好。前提是中文系统。
