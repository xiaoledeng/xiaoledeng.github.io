2014-05-05 11:00
#Book

1. 书的等级

  • 神书
  • 经典书
  • 一流书
  • 其他书

说明：

1.可以参考书籍的google Scholar上面的引用率，来考察一本书的影响，限于专业方面的书籍；

2.每个民族都有自己的经典书，可以参考经典书籍系列书；

3.最好推荐普世价值的书，不限于某种专业；

3.下面仅仅是一部分，后续待补充。

2.有关的介绍

  • 神书，少之又少。

经典书中经过时间的沉淀，为人们所承认。

遇到的有：《论语》、《圣经》、《如何阅读一本书》、《像艺术家一样思考》

  • 经典书

可以参考经典书籍书目，为各个专业的源头。

投资领域：《证券分析》

绘画：《素描的准则》

编剧：《故事》

积极心理学：《象与骑象人》、《超越自卑》、《少有人走的路》

编程：《计算机程序的构造和解释》、《计算机程序设计艺术》

数学：《数学与猜想》、《怎样解题》

英语口语：《美语发音13秘诀》

  • 一流书

经济学：曼昆的《经济学原理》，兹维·博迪的《金融学》，《门口的野蛮人》，

IT：《The C Programming Language》

电信：《TCP/IP详解》

摄影：《美国纽约摄影学院摄影教材》

漫画：《理解漫画》、《制造漫画》

绘画：《伯里曼人体结构》

心理学：《心理学与生活》，《幸福的方法》

  • 其他书

参考文档，有口碑的畅销书，理性的行业分析书籍等

看此类书的目的：

1）解决当前的问题

2）考试

3）开阔眼界

4）娱乐

5）并无可取，浪费时间
