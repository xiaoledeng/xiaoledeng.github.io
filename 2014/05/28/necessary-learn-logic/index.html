逻辑学习的必要性

2014-05-28
#Life #Ideas #Experiences

毛主席说过，逻辑是一门独立的学问，大家都要学点。

歌德说，守住时间，它总是在匆匆飞逝，我建议你们遵循一定的步骤来节约时间，为了达到这个目的，我首先推荐你们学习一门逻辑学课程。

自己，日常生活反思自己的行为，需要到逻辑这一经过无数先人贤哲不断完善的利器，李笑来25岁明白自己缺乏逻辑方面的锻炼，而我23岁明白了逻辑思维的力量，这一利器必须学会，而之前一直寻找的批判性思维很明显是属于逻辑学这一范围的，正如乾坤指出的，我缺乏这方面的东西，所以现在就要做到这方面的蜕变。

问题：为什么要学习逻辑学？怎样系统化学习逻辑学？如何在日常生活学习工作中做到有逻辑地做事，说话，写作，交流？最后一点最重要。

逻辑的范围，历史，发展的不足之处。经典古典逻辑，现代逻辑，关键词：批判性思维，理性，逻辑，博弈。

一直以来，自己能够自发的进行学习，践行终身学习之道。我相信随着不断得学习、积累感悟，终有一天能够得到真正的智慧。每次遇到新的知识、事物都能追本溯源，通过各种手段来了解详情。

Logic这个词很早就知道，然而在中国的教育体系中，没有普及这门知识体系中必不可少的课程，那就需要自己去学习。

反思个人日常生活中的行为，特别是跟人交流的时候，思维老是跳跃几阶，给交谈者造成困惑，给交流造成来困难，这是一两年前的情况，自己意识到这个问题，现在慢慢改变。甚至在某次很重要的会议上，出了一个大丑，用现在逻辑学里面的一个术语来归纳就是：Begging the Question，把尚须证明的结论当作不证自明的条件，举个例子：因为他在说谎，所以他是一个说谎的人。朋友在毕业之前的赠言是希望我能好好学习逻辑。在理性思考、批判性思维方面和逻辑思维等方面，自己很欠缺，需要不断打磨自己。

如何系统地学习逻辑学？

  1. 看原版书、做笔记；
  2. 融会贯通相关知识点；
  3. 列两份list，分别为逻辑思维的步骤和应该避免的逻辑陷阱；
  4. 不断的实践，吸取他人经验，总结自己习惯，不断完善。

自己应该学习到什么程度？

  1. 在日常生活、学习、工作中，能够做到有逻辑地做事情、有逻辑地与人交流言谈、有逻辑地写作；
  2. 成为一个思想自由地人。
